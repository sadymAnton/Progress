Справочник "Вакансии"

Справочник предназначен для хранения информации о вакансиях компании, включая подробное описание требований, предъявляемых к кандидату, его должностных обязанностей и условий приема на работу. В зависимости от выбранного режима кадрового планирования вакансии вводятся по центрам ответственности или по структуре юридических лиц. Режим кадрового планирования выбирается в обработке Кадровое планирование.

Вакансии обладают рядом дополнительных свойств, которые нельзя указать для строк кадрового плана. Такими свойствами являются:

  • Ответственный — ответственный за вакансию; по умолчанию в качестве значения реквизита указывается текущий пользователь;
  • Заявитель — автор заявки, пользователь, инициировавший открытие вакансии;
  • Плановая дата закрытия — плановая дата закрытия вакансии;
  • Закрыта — после установки флажка становится доступно поле фактической даты закрытия вакансии.
