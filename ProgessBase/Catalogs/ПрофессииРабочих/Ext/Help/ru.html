Справочник «Профессии рабочих»

Справочник используется при вводе кадровых данных о сотрудниках организаций.

Справочник может быть заполнен по данным раздела "Профессии рабочих" Общероссийского классификатора профессий рабочих, должностей служащих и тарифных разрядов ОК 016-94. Для этого предназначена кнопка «Подбор из ОКПДТР» в форме списка.
