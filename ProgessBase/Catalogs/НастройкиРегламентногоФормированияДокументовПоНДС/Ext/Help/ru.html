Справочник «Настройки регламентного формирования документов по НДС»

Справочник предназначен для хранения настроек формирования документов регламентных документентов по учету НДС.

Формирование документов по настройке может выполняться

  • в обработке «Формирование документов по НДС»;
  • автоматически, регламентным заданием.


При выполнении регламентного задания автоматически заполняются и проводятся следующие документы:

  • «Начисление НДС по СМР (хозспособом)»;
  • «Распределение НДС косвенных расходов»;
  • «Восстановление НДС»;
  • «Формирование записей книги покупок» (в том числе, по вычетам при реализации по ставке 0%);
  • «Формирование записей книги продаж» (в том числе, по реализации по ставке 0%);

Расписание

Для автоматического формирования документов следует установить флаг «Использовать расписание» и ввести расписание.

В момент записи редактируемой Настройки будет создано связанное с ней регламентное задание.

При каждом запуске по расписанию формирование документов на конец месяца, соответствующего текущей дате.
