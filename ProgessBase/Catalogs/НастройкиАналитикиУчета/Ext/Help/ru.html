Справочник «Настройки аналитики учета»

Справочник «Настройки аналитики учета» предназначен для просмотра и изменения настройки детализации затрат при использовании расширенной аналитики учета затрат.

Настройка доступна только для следующих режимов использования расширенной аналитики:

  • «Регламентированный учет с дополнительной аналитикой»
  • «Управленческий и регламентированный учет».

При этом можно настраивать только некоторые разрезы аналитики учета затрат. Флажки, соответствующие другим разрезам - недоступны.

  • Для режима «Регламентированный учет с дополнительной аналитикой» можно настраивать детализацию затрат только по регл. учету
  • Для режима «Управленческий и регламентированный учет» - только по упр. учету

Внимание! Не рекомендуется изменять настройки детализации учета затрат если в базе существуют проведенные документы позже даты перехода на расширенную аналитику.
