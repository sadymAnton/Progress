Справочник «Группы писем электронной почты»

Cправочник предназначен для хранения групп писем электронной почты для каждой учетной записи и используется во встроенном почтовом клиенте конфигурации.

Справочник подчинен справочнику «Учетные записи электронной почты». Все электронные письма системы можно хранить по группам в каждой учетной записи. Наглядное отображение групп электронных писем показано на форме  основной консоли встроенного почтового клиента.

При создании нового элемента справочника «Учетные записи электронной почты» в справочнике «Группы писем электронной почты» создаются четыре элемента, группы электронных писем по умолчанию для новой учетной записи:

  • Входящие;
  • Исходящие;
  • Черновики;
  • Удаленные.

При необходимости пользователь может создавать новые группы, подчинять группы друг другу (справочник организован по принципу иерархии элементов).
