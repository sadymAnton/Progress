Справочник «Маршруты согласования заявок»

Предназначен для хранения маршрутов согласования заявок на расходование денежных средств.

Маршрут может содержать несколько этапов согласования за счет использования иерархии. В этом случае процесс согласования будет происходить вверх по иерархии.

Для каждого этапа согласования указывается список пользователей, которые могут согласовывать заявку на данном этапе.
