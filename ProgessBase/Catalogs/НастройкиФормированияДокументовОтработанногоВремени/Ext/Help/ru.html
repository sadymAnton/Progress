Справочник «Настройки формирования документов отработанного времени»

Справочник предназначен для хранения настроек формирования документов отработанного времени по данным документов  «Отчет о составе смены».

По настройке будут сформированы документы  «Табель учета рабочего времени организации».

Формирование документов может выполняться

  • в обработке «Формирование документов по данным оперативного учета производства»
  • автоматически, регламентным заданием

В поле «Начиная с» указана дата, до которой данные документов «Отчет о составе смены» учтены в сформированных документах.
Её значение

  • изменяется автоматически по мере формирования документов
  • может быть изменено вручную (по кнопке выбора даты).

Параметры создаваемых документов

Организация

Документы будут сформированы по указанной организации. Если организация не указана, тогда будут сформированы документы по всем организациям, в которых работают физические лица введенные в документ «Отчет о составе смены».  

Подразделения организаций

Если подразделение организаций заполнено, то в табличную часть «Исполнители» документа «Табель учета рабочего времени организации» попадут все сотрудники указанного подразделения. Если поле не заполнено, то тогда в табличную часть попадут все сотрудники той организации, которая указана в шапке документа. Нужно отметить, что реквизит «Подразделение организаций» из выбираемой настройки может не совпадать с реквизитом «Подразделение» документа «Отчет о составе смены». Если физическое лицо работать по управленческому учету в одном подразделении, а по регламентированному в другом, то в случае, если в настройке указано такое регламентированное подразделение, то такие физические лица попадут в документ «Табель учета рабочего времени организации».  

Детализация

Документы будут сформированы с заданной детализацией. Каждый документ будет включать данные за период

  • не больший, чем период детализации
  • относящийся к одному календарному месяцу

Документы формируются датой последнего дня периода, данные за который содержат.

Способ ввода времени

Для детализации со значением «Месяц» можно задать способ ввода времени:

  • По дням периода
  • Сводно, в целом за период

Указанный способ ввода времени будет установлен в документе «Табель учета рабочего времени организации».

Краткое описание механизма формирования документов отработанного времени

При выполнении формирования документов отработанного времени будут обрабатываться документы «Отчет о составе смены» за выбранный период. Период обработки документов будет определен как:

  • В толстом клиенте: период указанный в обработке  «Формирование документов по данным оперативного учета производства»
  • В тонком клиенте: период с даты указанной в реквизите «Начиная с» выбранной настройки из справочника «Настройки формирования документов отработанного времени» по дату окончания интервала дат просмотра списка документов мастера смены или если эта дата не заполнена, то по рабочую дату.

В табличную часть документа «Табель учета рабочего времени организации» попадут все физические лица из выбранных документов  «Отчет о составе смены» за период, которые работают в указанном подразделении организации из выбранной настройки, или если подразделение не указано, то те, которые работают в указанной организации. Если физическое лицо из документа «Отчет о составе смены» работает в нескольких организациях, то такое физическое лицо (соответствующий этому физическому лицу элемент справочника «Сотрудники») попадет в документ «Табель учета рабочего времени организации» по каждой такой организации.

При заполнении табличной части документа «Табель учета рабочего времени организации» отработанные дни будут заполнены как «Я 8» без учета ночных и вечерних часов. Выходные дни будут отмечены как «В», а дни неявок как «НН 8». За неявку принимается следующие случаи:

  • Если в документе «Отчет о составе смены» в табличной части у физического лица не установлен флаг «Явка»
  • Если в документе «Отчет о составе смены» в табличной части не указано физическое лицо, которое должно было работать по графику работы.
