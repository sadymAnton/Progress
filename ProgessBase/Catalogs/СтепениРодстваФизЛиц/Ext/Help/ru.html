Справочник «Степени родства физических лиц»

Справочник используется при вводе кадровых данных о сотрудниках организаций. 

Справочник может быть заполнен по данным фасета "Родство, свойство" из Общероссийского классификатора информации о населении ОК  018-95. Для этого предназначена кнопка «Подбор из ОКИН» в форме списка. При этом открывается список фасета "Родство, свойство", в котором можно найти, выбрать и занести в справочник нужную степень родства.
