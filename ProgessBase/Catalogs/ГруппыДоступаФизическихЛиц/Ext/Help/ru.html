Справочник "Группы доступа физических лиц"

Справочник используется для настройки доступа к физическим лицам.

 

Группы доступа создаются плоским списком или иерархическим. В иерархическом списке физические лица вложенных групп неявно входят в группы-родители.

 

Группы могут формироваться по структуре предприятия, по должностям и проектным командам, по видам договора, например:

  • Администрация
  • Отдел закупок
  • Отдел продаж
  • Сотрудники предприятия
  • Договорники
  • Кандидаты на работу

Сотрудник может входить только в одну группу доступа.

 

Для каждой группы доступа можно указать следующие ограничения доступа:

  • Просмотр списка физических лиц
  • Просмотр данных физических лиц
  • Редактирование данных физических лиц

 

Право на просмотр списка физических лиц можно предоставлять большинству пользователей, в то время как право на просмотр и редактирование данных физических лиц целесообразно предоставлять только ответственным сотрудникам.
