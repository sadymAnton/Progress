Справочник «Способы распределения затрат»

В справочнике задаются способы распределения затрат.

Каждый способ распределения характеризуется базой распределения.

База распределения определяет

  • в какой пропорции
  • на какую аналитику

должны быть распределены затраты.

Особенности использования отдельных способов распределения

Группа способов «На выпуск»

При расчете базы распределения учитываются услуги

  • либо оказание которых отражено в системе документом «Акт об оказании производственных услуг»
  • либо для которых в документе «Установка параметров учета номенклатуры» задана аналитика затрат

Способы «По объему продаж» и «По выручке»

При расчете базы распределения учитывается только та продукция, которая выпускалась в текущем месяце

Способы не применяются, если затраты связаны с оказанием услуг и при этом учитываются в разрезе заказов.

Добавление и изменение способов распределения

При добавлении способа распределения следует описать правила получения (расчета) базы.

База может быть

  • либо задана явно, в виде таблицы (тип базы – «табличный»)
  • либо рассчитываться по данным информационной базы (тип базы – «по данным ИБ»)

Тип базы определяется при вводе нового элемента справочника.

База распределения в виде таблицы

Как правило, стоит применять, если база статичная, не меняется со временем.
Базу распределения следует задать в форме элемента справочника – заполнить таблицу.
Состав колонок таблицы зависит от направления распределения затрат, выбранного в поле «Куда распределять».

Настройка расчета базы распределения «По данным ИБ»

Для настройки расчета следует задать схему компоновки данных.

База распределения рассчитывается при проведении документа «Расчет себестоимости».
Для расчета базы выполняется запрос к ИБ, заданный с помощью схемы компоновки данных.

Схемы компоновки данных рекомендуется разрабатывать, используя в качестве образца способы распределения, которыми заполнен справочник.

Настройка «Изменять способы распределения затрат»

Если используется настройка Изменять способы распределения затрат, то следует учитывать особенности при настройке и использовании способов распределения затрат:

  1. Не применяются предопределенные способы

    • По выручке от реализации услуг
    • Перенести на производственные расходы
    • Директ-костинг
    • Расходы на продажу

  2. Не применяются способы распределения с типом базы

    • Табличный
    • По данным ИБ

  3. Не применяются способы распределения с направлением

    • На затраты
    • На результат

  4. При настройке способа распределения следует задать правила расчета базы на закладках «База распределения», «Отбор», «Фильтры»

Закладка «База распределения»

На закладке «База распределения» определяются основные правила, по которым будет рассчитываться база распределения затрат:

  • База распределения. Выбирается стратегия получения базы распределения, то есть выполняется оценка объема продукции, на который относятся затраты, и рассчитывается доля затрат на каждый вид продукции. Возможны следующие варианты:
    • По объему выпуска. В качестве базы распределения используется объем выпущенной продукции. Оценка объема выпуска выполняется в соответствии со значением показателя базы распределения: в натуральных единицах измерения, в ценах номенклатуры (в соответствующем реквизите выбирается тип цен), в фактических ценах реализации.
    • По объему продаж. В качестве базы распределения используется объем продаж продукции. Оценка объема продаж выполняется в соответствии со значением показателя базы распределения: в натуральных единицах измерения, в ценах номенклатуры (в соответствующем реквизите выбирается тип цен), в фактических ценах реализации.
    • По основному сырью. База распределения соответствует объему вхождения некоторого основного сырья в продукцию. Основное сырье указывается в соответствующем реквизите. Основное сырье оценивается либо в натуральных единицах измерения, либо в выбранных ценах номенклатуры.
    • По нормативам. В качестве базы распределения используется плановая стоимость продукции умноженная на объем выпущенной продукции. Плановая стоимость продукции задается в регистре сведений «Плановая себестоимость номенклатуры».
    • Вручную. База распределения задается вручную в документе «Установка базы распределения затрат».
  •  Показатель базы распределения. Используется только при выборе баз распределения «По объему выпуска», «По объему продаж» или «По основному сырью». В реквизите можно выбрать следующие значения:
    • Натуральные ед. изм. – база рассчитывается в натуральных единицах измерения;
    • Цены номенклатуры – база рассчитывается по стоимости в выбранных ценах номенклатуры;
    • Фактическая цена реализации – база рассчитывается по фактической цене реализации. Если используется база распределения «По объему выпуска», то рассчитывается средняя фактическая цена реализации и умножается на объем выпуска.
  • Распределять на собственную продукцию. Признак распределения косвенных затрат на собственную готовую продукции, услуги.
  • Распределять на продукция стороннего переработчика. Признак распределения косвенных затрат на готовую продукцию, выпущенную сторонним переработчиком.
  • Распределять на продукцию из давальческого сырья. Признак распределения косвенных затрат на готовую продукцию, выпущенную из давальческого сырья.
  • Распределять на наработку. Признак распределения косвенных затрат на объем наработки.
  • Распределять на подчиненные подразделения. Признак необходимости расчета базы распределения с учетом подчиненных подразделений. Для общехозяйственных затрат на предприятиях могут использоваться разные базы распределения по подразделениям и внутри подразделения по продукции. Поэтому в отдельном реквизите «Способ распределения по подразделениям» указывается способ распределения затрат между подчиненными подразделениями.
  • Изменить на процент. Увеличение / уменьшение рассчитанной базы на указанный процент.
  • Изменить на коэффициент. Увеличение / уменьшение рассчитанной базы на указанную величину.
  • Округлить до. Задает порядок округления базы распределения до указанного значения.

Закладка «Отбор»

На закладке «Отбор» устанавливается отбор значений  полей при определении базы распределения. Можно  установить отбор по следующим полям:

  • Вид затрат;
  • Статус материальных затрат;
  • Характер затрат;
  • Статья затрат;
  • Подразделение – имеет смысл только для общехозяйственных затрат;
  • Подразделение организации – имеет смысл только для общехозяйственных затрат.

Закладка «Фильтры»

При распределении затрат могут налагаться ограничения на выбор номенклатурных групп и продукции. На закладке выбирается сначала тип фильтра, а затем заполняется табличная часть закладки:

  • Номенклатурная группа – колонка видна при выборе типа фильтра «Номенклатурные группы»;
  • Продукция – колонка видна при выборе типа фильтра «Продукция»;
  • Коэффициент – корректировка базы распределения по указанной номенклатурной группе / продукции на указанную величину.
