Справочник «Способы отражения расходов»

Справочник содержит способы отнесения на затраты предприятия расходов:

  • по амортизации ОС и НМА,
  • при включении стоимости ОС в состав расходов при принятии к налоговому учету,
  • при погашении стоимости переданной в эксплуатацию спецодежды, спецоснастки.

В справочнике указываются способы получения базы для распределения расходов, счета и аналитика для отнесения на затраты по управленческому, бухгалтерскому и налоговому учету.

Перечень способов задается в табличной части «Способы». Каждая строка табличной части определяет совокупность параметров для отнесения расходов по амортизации на затраты. В том случае если введено несколько строк, расходы по амортизации вначале будут распределены по строкам (способам) согласно коэффициентам распределения. Коэффициенты распределения указываются в каждой строке. В случае если в табличной части указана только одна строка (т.е. задан только один способ), коэффициент следует устанавливать равный единице.

После распределения расходов по способам (согласно заданным коэффициентам), для тех способов, где в статье затрат указано отнесение на нематериальные производственные затраты, сумма расходов по амортизации может быть дополнительно распределена по базе. Эта база определяется в реквизите Способ распределения затрат на выпуск.

Для основных средств способ отражения расходов по амортизации может быть установлен в документах «Принятие к учету ОС», «Изменение способов отражения расходов по амортизации». Для нематериальных активов - в документах «Принятие к учету НМА организаций», «Изменение способа отражения расходов по амортизации НМА организаций». Для спецодежды, спецоснастки - в справочнике «Назначения использования».
