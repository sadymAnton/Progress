Справочник "Конвертации из информационных баз 1С:Предприятия 7.7"

Справочник "Конвертации из информационных баз 1С:Предприятия 7.7" предназначен для хранения сведений о конфигурациях на платформе 1С:Предприятие 7.7, для которых поддерживается автоматический перенос данных при помощи специализированной обработки "Перенос данных из информационных баз 1С:Предприятия 7.7".

В форме списка справочника выводится информация о наименовании конфигурации, ее внутреннем идентификаторе, номере релиза конфигурации (перенос поддерживается только для релизов конфигураций не ниже указанного в справочнике) а также имена файлов выгрузки данных и правил конвертации.

По кнопке "Записать файлы конвертации на диск" производится сохранение файлов обработки конвертации данных и файла правил для выбранной конфигурации. Файлы сохраняются с теми именами, которые указаны в соответствующих колонках списка.

Если предполагается производить выгрузку данных из конфигураций на платформе 1С:Предприятие 7.7 при помощи универсальной обработки выгрузки данных, рекомендуется предварительно сохранять файлы конвертации из справочника "Конвертации из информационных баз 1С:Предприятия 7.7" в каталог "ExtForms" каталога информационной базы, из которой предполагается выгружать данные.
