Справочник "Способы отражения зарплаты в управленческом учете"

В справочнике описываются все возможные комбинации статей затрат, номенклатурных групп и объектов строительства. Введенные комбинации используются при отражении зарплаты сотрудников по центрам ответственности.

 

Несколько способов отражения являются предопределенными:

  • Отражение начислений по умолчанию
  • Не отражать в упручете

Способ «Отражение начислений по умолчанию» используется программой автоматически, когда при отражении по центрам ответственности для начислений не задан способ отражения.

Способ «Не отражать в упручете» имеет смысл указывать для начислений и удержаний, отражение в учете которых не требуется. В учете не отражаются как сами начисления, для которых задан данный способ отражения, так и налоги и взносы рассчитанные с этого начисления.

 

Способы отражения в учете могут быть указаны:

  • для основных и дополнительных регламентированных начислений, и управленческих начислений может быть заполнен реквизит «Отражение в упручете» формы вида начисления
  • для всех управленческих начислений сотрудника, регистрация способа отражения производится документом «Учет заработка сотрудников»

 

При отражении в учете данные об отражении в учете используются в следующей последовательности:

  • Если указан способ отражения в учете конкретного начисления – используется этот способ
  • Если указано отражение в учете заработка конкретного сотрудника – используется этот способ
  • Если для начисления ничего не указывалось, используется способ «Отражение начислений по умолчанию»
