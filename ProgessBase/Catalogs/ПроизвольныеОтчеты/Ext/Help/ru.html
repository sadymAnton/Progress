Справочник "Произвольные отчеты"

Справочник предназначен для создания новых отчетов.

Создание и изменение отчетов доступно пользователям с ролью "Администрирование дополнительных форм и обработок". Также таким пользователям доступен обмен отчетами (подменю "Обмен").

Для нового отчета необходимо создать схему компоновки данных (меню "Схема компоновки данных").

Если схема компоновки содержит наборы данных "Объект", то для них необходимо настроить источники данных. Список настроек источников данных отображается в поле "Источники данных". Если источник данных соответствует несуществующему набору данных, то он выделяется красным цветом. Такой источник данных необходимо либо удалить, либо указать другой набор данных.

 

Для отчета можно использовать "Конструктор настройки показателя монитора эффективности".
