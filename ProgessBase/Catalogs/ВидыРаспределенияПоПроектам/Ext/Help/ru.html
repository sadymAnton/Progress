Справочник «Виды распределения по проектам»

В справочнике задается стратегия распределения затрат по проектам. Это касается таких затрат как издержки обращения, коммерческие расходы, общепроизводственные расходы, общехозяйственные расходы, прочие. Распределение затрат по проектам выполняется только по управленческому учету.

 

Метод распределения определяет момент времени распределения затрат по проектам. Значение «Регламентным документом» означает, что затраты будут распределяться по проектам при проведении документа «Расчет себестоимости». Значение «При проведении документов» означает, что затраты будут распределяться по проектам непосредственно при проведении документа, которым отражаются затраты.

 

Коэффициенты распределения затрат необходимо задать с помощью документа «Установка долей распределения по проектам».
