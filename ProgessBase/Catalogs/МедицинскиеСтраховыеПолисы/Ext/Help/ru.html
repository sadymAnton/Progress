Справочник "Медицинские страховые полисы"

Справочник предназначен для хранения информации о медицинских страховых полисах обязательного и добровольного страхования.

 

Обязательное страхование

Полисы можно оформлять только на сотрудников, работающих в указанной организации на дату выдачи полиса.

Если у полиса не указана дата окончания, такой полис считается бессрочным.

Пол, дата рождения, удостоверение личности, адрес проживания и домашний телефон заполняются автоматически из личных данных сотрудника и запоминаются в полисе. Заполненную автоматически информацию можно изменить. Обратите внимание, при изменении информации в карточке личных данных сотрудника, введенная в полисе информация изменена не будет.

Список сотрудников, которым необходимо оформить, продлить или прекратить полис, можно получить в отчете "Медицинское страхование".

 

Добровольное страхование

Полисы добровольного страхования можно оформлять на любых сотрудников, в том числе и на тех, которые не работают в организации.

Для полиса добровольного страхования можно указать программу страхования. Список программ страхования определяется пользователем в одноименном справочнике.

Полис добровольного страхования можно оформить и на родственника сотрудника. Для этого достаточно указать ФИО родственника, или выбрать родственника из списка родственников, введеных в карточке личных данных сотрудника.

При оформлении полиса на родственника, пол, дату рождения, удостоверение личности, адрес проживания и домашний телефон нужно указывать для родственника, а не для сотрудника.

Список полисов добровольного страхования для передачи в ДМС можно получить в отчете "Медицинское страхование".

 

К полисам можно прикреплять дополнительные файлы.
