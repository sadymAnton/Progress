Документ «Установка долей распределения по проектам»

Для таких затрат как издержки обращения, коммерческие расходы, общепроизводственные расходы, общехозяйственные расходы, прочие можно задать порядок распределения по проектам.

В шапке документа выбирается «Вид распределения». Это элемент справочника «Виды распределения по проектам».

В табличной части указываются коэффициенты распределения по проектам.

При проведении документа данные записываются в регистр сведений «Распределение по проектам», на основании данных которого производится распределение по проектам и формируются движения с указанными проектами по результатам распределения.
