Документ «Корректировка прочих затрат»

Документом отражается операция корректировки нематериальных затрат, включающая в себя изменение подразделения, номенклатурной группы, заказа, статьи затрат, счета учета.

Документ "Корректировка прочих затрат" рекомендуется использовать в тех случаях, когда порядок распределения косвенных затрат не может быть воспроизведен документами конфигурации или воспроизведен неверно. В таких случаях проводится ручной расчет и результаты отражается в учете с помощью этого документа

 

В шапке документа можно выбрать следующие реквизиты:

  • Подразделение. Подразделение, по которому производится корректировка нематериальных затрат. Значение реквизита выбирается из справочника «Подразделения». Реквизит является обязательным для заполнения.
  • Подразделение организации. Подразделение организации, по которому производится корректировка нематериальных затрат. Значение реквизита выбирается из справочника «Подразделения организаций». Реквизит является обязательным для заполнения.

 

Затраты

 

На закладке указываются нематериальные затраты , по которым производится корректировка, сумма затрат и аналитика учета затрат на момент корректировки:

  • Статья затрат. Исходная статья затрат, по которой были отражены нематериальные затраты. Значение реквизита выбирается из справочника «Статьи затрат».
  • Заказ. Исходный заказ, по которому производится корректировка. Значение реквизита выбирается из списка документов «Заказ покупателя» или «Заказ на производство».
  • Аналитика. Аналитика учета затрат на момент заполнения документа, вид аналитики определяется выбранной статьей затрат. Вид аналитики справочно показывается в реквизите «Вид аналитики».
  • Сумма. Сумма корректировки затрат в валюте управленческого учета.
  • Сумма (БУ). Сумма корректировки затрат для бухгалтерского учета. Значение реквизита рассчитывается автоматически при изменении суммы в валюте упр. учета.
  • Сумма (НУ). Сумма корректировки затрат для налогового учета. Значение реквизита рассчитывается автоматически при изменении суммы в валюте упр. учета.
  • Счет затрат (БУ). Исходный счет бухгалтерского учета затрат. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе статьи затрат.
  • Счет затрат (НУ). Исходный счет налогового учета затрат. Значение реквизита выбирается из плана счетов налогового учета и заполняется автоматически при выборе статьи затрат или выборе счета затрат по бухгалтерскому учету. 

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». При нажатии кнопки «Заполнить по остаткам» открывается список выбора характера затрат. После выбора в табличную часть документа будут подобраны остатки затрат по выбранному характеру затрат. При необходимости в списке выбора можно указать вариант "<Все>" - в этом случае заполнение будет выполнено без отбора по характеру затрат.

 

Получатели

 

На закладке указываются новые значения аналитики затрат. Каждая строка закладки «Получатели» является расширением соответствующей строки закладки «Затраты». Ввод новых строк, удаление, сортировка на данной закладке не доступны. Новые значения аналитики затрат можно предварительно заполнить исходными значениями, нажав кнопку «Заполнить» - «Заполнить по данным о затратах». Тогда новые значения подразделения, заказа, аналитики, счета затрат будут заполнены соответствующими данными с закладки «Затраты», либо данными из шапки документа. Далее значения аналитики затрат можно откорректировать вручную.

  • Подразделение. Подразделение - получатель затрат. Значение реквизита выбирается из справочника «Подразделения».
  • Подразделение организации. Подразделение организации - получатель затрат. Значение реквизита выбирается из справочника «Подразделения организаций».
  • Заказ. Указывается новый заказ покупателя или заказ на производство, по которому отражаются затраты. Значение выбирается из списка документов «Заказ покупателя» или «Заказ на производство».
  • Аналитика. Новое значение аналитики затрат. Вид аналитики определяется выбранной статьей затрат. Вид аналитики справочно показывается в реквизите «Вид аналитики».
  • Статья затрат. Новая статья затрат, по которой будут отражены затраты. Значение реквизита выбирается из справочника «Статьи затрат».
  • Счет затрат (БУ). Новый счет бухгалтерского учета затрат. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе статьи затрат.
  • Счет затрат (НУ). Новый счет налогового учета затрат. Значение реквизита выбирается из плана счетов налогового учета и заполняется автоматически при выборе статьи затрат или выборе счета затрат по бухгалтерскому учету.
