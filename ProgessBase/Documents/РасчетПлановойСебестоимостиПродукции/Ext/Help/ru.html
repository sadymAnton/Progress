Документ «Расчет плановой себестоимости продукции»

Документ предназначен для расчета плановой себестоимости продукции и сохранения результатов расчета в информационной базе.

Для расчета плановой себестоимости используются данные:

  • спецификаций номенклатуры: количество требуемых для выпуска единицы продукции затрат (материалов, работ и т.п.)
  • цен номенклатуры: стоимость единицы каждой затраты (материала, работы и т.п.), зарегистрированной в системе

Закладка «Параметры расчета»

Указываются параметры расчета плановой себестоимости продукции:

  • Отбор номенклатуры, для которой следует рассчитать плановую себестоимость
  • Тип цен для оценки затрат
  • Процент накладных расходов, на который следует изменить результат расчета

Закладка «Себестоимость»

Содержит сведения о рассчитанной плановой себестоимости продукции:

  • Продукция, Характеристика продукции
  • Спецификация продукции
  • Единица измерения и Количество продукции на которые рассчитана себестоимость (заполняются автоматически из спецификации, недоступны для изменения)
  • Себестоимость

Закладка «Состав затрат»

Содержит сведения о составе прямых затрат на выпуск продукции:

  • Продукция, Характеристика, Спецификация продукции, к которым относится затрата
  • Затрата, Характеристика затраты
  • Единица измерения и Количество затраты
  • Цена и Сумма затраты

Автоматическое заполнение документа

Документ можно заполнить автоматически с помощью кнопки «Рассчитать».

При этом документ будет заполнен в соответствии с данными, указанными на закладке «Параметры расчета»:

  • Список продукции на закладке "Себестоимость" заполняется в соответствии с отбором. Используется основная спецификация продукции
  • По данным спецификаций заполняется закладка "Состав затрат": рассчитывается количество ресурсов, требуемых для выпуска указанного количества продукции
  • В соответствии с типом цен выполняется суммовая оценка прямых затрат
  • Плановая себестоимость продукции рассчитывается как сумма прямых затрат, измененная на процент накладных расходов
