Документ "Лист 1А"

Документ содержит все необходимые сведения для формирования листа Раздела 1А уведомления о контролируемых сделках. При печати или выгрузке уведомления для каждого документа будет сформирован один лист Раздела 1А, один лист Раздела 2 или Раздела 3, а также один или несколько листов Раздела 1Б (для каждой строки в табличной части "Листы 1Б" будет сформирован отдельный лист).

Документы автоматически формируется при заполнении уведомления, но пользователь затем их может скорректировать.

На закладке "Информация о сделке" указываются показатели контролируемых сделок, которые содержатся в листе Раздела 1А

  • Группа "Основание отнесения сделки к контролируемой" - определяет значения показателей 121-124
  • Группа "Особенности отнесения сделки к контролируемой" - определяет значения показателей 131-135
  • Способ определения цены сделки - определяет значения показателей 220-259
  • Группа "Суммы по сделкам" - содержит фактические суммы полученных доходов или произведенных расходов, определяемых в соответствии с правилами бухгалтерского учета. Отдельно указываются суммы доходов и расходов по сделкам, цены которых подлежат регулированию. Суммы доходов и расходов расчитываются автоматически как сумма показателей 140 листов Раздела 1Б. Однако, если суммы контролируемых сделок не совпадают с доходами или расходами организаций по контролируемым сделкам, то возможно ручное редактирование этих показателей. Для этого нужно выбрать флаг Редактировать суммы сделок

На закладке "Предмет сделки" указываются показатели, относящиеся к предмету сделки: наименование предмета, тип и коды ТН ВЭД, ОКП и ОКВЭД.

На закладке "Листы 1Б" в табличной части указывается список листов Раздела 1Б, которые относятся к данному листу Раздела 1А

Значения показателей листов Разделов 1А и 1Б в документе указываются с точностью, которая соответствует точности первичных документов. При печати или выгрузке в файл они будут округлены до целых значений в соответствии с форматом ФНС. Для таких показателей рядом с полем ввода справочно указываются округленные значения.
