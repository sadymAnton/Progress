Документ «Реестр сведений в ФСС о ежемесячных пособиях по уходу»

Документ предназначен страхователям, участвующим в Пилотном проекте ФСС, для формирования реестра сведений, необходимых для назначения и выплаты пособия по уходу за ребенком до полутора лет, в соответствии с Постановлением Правительства РФ от 21.04.2011 № 294 по форме, утвержденной Приказом ФСС РФ от 17.06.2011 № 196.

Автоматическое заполнение документа

Меню Заполнить позволяет внести в перечень пособий все зарегистрированные со времени последней передачи реестра в ФСС проведенные документы «Отпуск по уходу за ребенком», в которых выплачивается пособие до полутора лет.

Совет. Чтобы после исправления данных в документе «Отпуск по уходу за ребенком» обновить сведения в Реестре, необходимо воспользоваться кнопкой Обновить.

При необходимости сведения для назначения и выплаты пособий могут быть дополнены или уточнены, либо полностью введены вручную.

Печатные формы

Кнопка Печать позволяет получить для вывода на принтер т.н. "Опись заявлений и документов" по форме, утвержденной Приказом ФСС РФ от 17.06.2011 № 195, а также печатную форму "Реестр сведений, необходимых для назначения и выплаты ежемесячного пособия по уходу за ребенком".

Формирование данных для передачи в ФСС

Кнопка Показать файл позволяет сформировать файл для просмотра. Кнопка Записать файл на диск позволяет скопировать этот файл непосредственно на диск (включая съемные носители).

При формировании файлов или печатных форм автоматически проводятся проверки выводимых данных и в случае их недостаточности либо ошибочности выдаются диагностические сообщения.

Проведение документа

Проведение документа (кнопка ОК) означает, что данные переданы в ФСС.
