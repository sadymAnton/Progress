Документ «Расчет себестоимости»

Документ выполняет следующие функции:

  • суммовую оценку стоимости движения МПЗ за месяц
  • расчет себестоимости готовой продукции и услуг
  • распределение затрат, не включаемых в себестоимость продукции и услуг (для регламентированного учета)

Поведение документа - разное, зависит от установленного в настройках параметров учета режима учета затрат.

Установлен режим использования расширенной аналитики учета затрат

В режиме использования расширенной аналитики учета затрат, документ может быть проведен по одному из учетов

  • управленческому
  • регламентированному (бухгалтерскому и налоговому)

Если указана настройка закрытия месяца (заполнен реквизит документа), то расчет баз и распределение расходов выполняется в соответствии с этой настройкой.

В списке "Выполняемые действия" можно указать список действий, которые будут выполняться при проведении документа. Полный список действий (перечислены в том порядке, в котором выполняются):

  • Распределение транспортно-заготовительных расходов
  • Определение списка реализованных услуг
  • Расчет базы распределения расходов
  • Распределение расходов по базе
  • Расчет фактической себестоимости
  • Формирование движений по регистрам

В списке не имеет смысла менять порядок этих действий. В принципе, можно удалить любое из действий. Однако в общем случае делать это не рекомендуется.

Далее – краткие пояснения по каждому из действий.

Распределение транспортно-заготовительных расходов

Действие выполняется, если их учет ведется на отдельном счете. 
В базу распределения включается и тара, которая учитывается на балансовых счетах (10.04, 41.03).

Определение списка реализованных услуг

Список услуг, себестоимость которых будет рассчитана, определяется по документам «Реализация товаров и услуг». Для каждой услуг определяется аналитика затрат – по данным закладки «Услуги» документа «Настройка параметров учета номенклатуры»

Расчет базы распределения расходов

Действие выполняется, если не используются способы распределения

  • Директ-костинг
  • Перенести на производственные расходы
  • По выручке от реализации услуг
  • Расходы на продажу  
  • с типом базы «Табличная»
  • с типом базы «По данным ИБ»

Производится расчет всех баз, по которым будут распределены расходы.

Рассчитанные базы записываются в регистры сведений «База распределения затрат» и «База распределения затрат (бухгалтерский учет)».

Распределение расходов по базе

Если базы записаны в регистры сведений (см. действие «Расчет базы распределения расходов»), то оцененные расходы распределяются по рассчитанным базам.

Если базы не записаны в регистры сведений, то рассчитываются базы и по ним распределяются все расходы

  • прямые
  • косвенные
  • издержки обращения
  • коммерческие
  • по методу «директ-костинг»

Расчет фактической себестоимости

Выполняется суммовая оценка стоимости движения МПЗ, в том числе:

  • использованных материалов
  • выпущенной продукции
  • реализованных товаров

Формирование движений по регистрам

Формируются движения по регистрам

  • бухгалтерии (для регл. учета)
  • стоимости основных средств (для упр.учета)

Установлен традиционный режим учета затрат

В традиционном режиме учета затрат документ может быть проведен только по одному виду учета. То есть для расчета себестоимости по всем видам учета необходимо сформировать несколько документов.

Последовательность этапов устанавливается пользователем в списке "Выполняемые действия".

Процедура закрытия месяца состоит из следующих этапов (выполняемых действий):

  • Расчет базы распределения производственных расходов - рассчитывается база распределения производственных расходов, распределяемых по заданному способу распределения. База рассчитывается для способов распределения, заданных для характера распределения "Производственные расходы";
  • Распределение производственных расходов - выполняется распределение расходов по характеру распределения "Производственные расходы" по рассчитанной базе;
  • Распределение продукции и затрат по переделам - определение номера передела для затрат и выпущенной продукции (услуг);
  • Расчет прямых затрат (по переделам или подразделениям):
    • списание стоимости материальных и нематериальных затрат из незавершенного производства, исходя из данных о распределении прямых затратах на выпуск,
    • включение стоимости прямых затрат в себестоимость выпуска продукции и формирование стоимости партий (отклонений) и услуг по прямым затратам,
    • корректировка себестоимости продукции по направлениям выпуска;
  • Определение продукции, исключаемой из базы распределения - производится определение продукции (услуг), которые должны быть исключены из базы распределения косвенных расходов (продукция, полностью списанная на распределяемые расходы);
  • Расчет базы распределения косвенных расходов - рассчитывается база распределения расходов, распределяемых по заданному способу распределения. База рассчитывается для способов распределения, заданных для характера распределения "Учитывать подразделение (косвенные расходы)" и "Не учитывать подразделение (косвенные расходы)";
  • Распределения косвенных затрат (по переделам или подразделениям):
    • распределение косвенных расходов по налоговому учету,
    • списание общепроизводственных, общехозяйственных расходов и расходов по браку в производстве в незавершенное производство,
    • распределение общепроизводственных, общехозяйственных расходов и расходов по браку в производстве на выпуск продукции,
    • корректировка себестоимости продукции с учетом косвенных затрат по направлениям выпуска;
  • Списание косвенных расходов на РБП - производится списание нераспределенных общепроизводственных и общехозяйственных расходов на расходы будущих периодов;
  • Списание общехозяйственных расходов по методу директ-костинг;
  • Расчет затрат встречного выпуска (по переделам или подразделениям) - расчет себестоимости выпущенной продукции (услуг) с учетом затрат встречного выпуска;
  • Списание издержек  обращения - списание издержек обращения на расходы текущего периода, в т.ч. распределение транспортных расходов;
  • Списание коммерческих расходов - списание коммерческих расходов на расходы текущего периода, в т.ч. распределение транспортных расходов;
  • Списание прочих затрат - списание прочих затрат по управленческому учету.

После выполнения всех расчетов можно произвести свертку движений по регистрам, которая позволяет минимизировать количество записей в регистрах бухгалтерского и налогового учета с одинаковой аналитикой.

На закладке "Дополнительно" указываются дополнительные настройки:

  • Флажок "Учитывать общепроизводственные расходы при распределении по переделам" - признак распределения продукции и затрат по переделам с учетом внутренних услуг (продукции) отражаемых в общепроизводственных расходах.
