Документ «Корректировка незавершенного производства»

Документом отражается операция корректировки незавершенного производства по материальным ценностям, включающая в себя изменение подразделения, номенклатурной группы, заказа, статьи затрат, счета учета.

 

В шапке документа можно выбрать следующие реквизиты:

  • Подразделение. Производственное подразделение, по которому производится корректировка незавершенного производства. Значение реквизита выбирается из справочника «Подразделения». Реквизит является обязательным для заполнения.
  • Подразделение организации. Производственное подразделение организации,  по которому производится корректировка незавершенного производства. Значение реквизита выбирается из справочника «Подразделения организаций». Реквизит является обязательным для заполнения.
  • Тип стоимости. Тип указания стоимости корректировки незавершенного производства. Можно выбрать следующие значения:
    • Рассчитывается. Стоимость корректировки незавершенного производства рассчитывается в конце месяца при расчете себестоимости выпуска.
    • Фиксированная. Стоимость корректировки незавершенного производства задается в документе и не корректируется при расчете себестоимости выпуска.

 

Материалы

 

В табличной части указываются материалы и полуфабрикаты, по которым производится корректировка незавершенного производства:

  • Номенклатура. Материал, по которому производится корректировка. Значение реквизита выбирается из справочника «Номенклатура».
  • Характеристика номенклатуры. Характеристика материала, если для  данного материала ведется учет по характеристикам. Значение реквизита выбирается из справочника «Характеристики номенклатуры».
  • Серия номенклатуры. Серия материала, если для  данного материала ведется учет по сериям. Значение реквизита выбирается из справочника «Серии номенклатуры».
  • Мест. Количество мест материала, по которому производится корректировка.
  • Единица. Единица измерения материала.
  • Количество. Количество материала в единице хранения остатков. Значение реквизита заполняется автоматически при указании количества мест.
  • Заказ. Исходный заказ, по которому производится корректировка. Значение реквизита выбирается из списка документов «Заказ покупателя» или «Заказ на производство».
  • Номенклатурная группа. Исходная номенклатурная группа, по которой производится корректировка.
  • Статья затрат. Исходная статья затрат, по которой были отражены материалы и полуфабрикаты в незавершенном производстве. Значение реквизита выбирается из справочника «Статьи затрат». В реквизите можно указывать только статьи с видом затрат «Материальные».
  • Цена. Цена корректировки. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма. Сумма корректировки незавершенного производства в валюте управленческого учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма (регл). Сумма корректировки незавершенного производства для бухгалтерского учета. Значение реквизита рассчитывается автоматически при изменении суммы в валюте упр. учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Сумма (НУ). Сумма корректировки незавершенного производства для налогового учета. Значение реквизита рассчитывается автоматически при изменении суммы в валюте упр. учета. Реквизит указывается, если выбран тип стоимости «Фиксированная».
  • Счет затрат (БУ). Исходный счет бухгалтерского учета незавершенного производства. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции в зависимости от вида подразделения организации.
  • Счет затрат (НУ). Исходный счет налогового учета незавершенного производства. Значение реквизита выбирается из плана счетов налогового учета и заполняется автоматически при выборе продукции в зависимости от вида подразделения организации.

 

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Возможны следующие варианты заполнения:

  •  Заполнить по остаткам. Заполнение производится по остаткам незавершенного производства. При заполнении по остаткам предыдущие значения строк в табличной части стираются.
  • Добавить из требования-накладной. В табличную часть добавляются данные из документов «Требование-накладная», которые выбирает пользователь в открывающемся диалоговом окне.

 

Получатели

 

На закладке указываются новые значения аналитики незавершенного производства. Каждая строка закладки «Получатели» является расширением соответствующей строки закладки «Материалы». Ввод новых строк, удаление, сортировка на данной закладке не доступны. Новые значения аналитики незавершенного производства можно предварительно заполнить исходными значениями, нажав кнопку «Заполнить»- «Заполнить по данным о материалах». Тогда новые значения подразделения, номенклатурной группы, статьи затрат, заказа, счета затрат будут заполнены соответствующими данными с закладки Материалы, либо данными из шапки документа. Далее значения аналитики незавершенного производства можно откорректировать вручную.

  • Подразделение. Подразделение - получатель затрат. Значение реквизита выбирается из справочника «Подразделения».
  • Подразделение организации. Подразделение организации - получатель затрат. Значение реквизита выбирается из справочника «Подразделения организаций».
  • Заказ. Указывается новый заказ покупателя, по которому отражаются затраты. Значение выбирается из списка документов «Заказ покупателя» или «Заказ на производство».
  • Номенклатурная группа. Новая номенклатурная группа, по которой отражается корректировка незавершенного производства.
  • Статья затрат. Новая статья затрат, по которой отражается корректировка незавершенного производства. Значение реквизита выбирается из справочника «Статьи затрат».
  • Счет затрат (БУ). Новый счет бухгалтерского учета, на котором отражаются затраты. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе статьи затрат.
  • Счет затрат (НУ). Новый счет налогового учета, на котором отражаются затраты. Значение реквизита выбирается из плана счетов налогового учета и заполняется автоматически при выборе статьи затрат или выборе счета затрат по бухгалтерскому учету.

 

Особенности проведения документа

 

Тип стоимости «Рассчитывается»

 

Данные табличных частей на закладках «Материалы» и «Получатели» отражаются в регистрах «Выпуск продукции» и «Затраты на выпуск продукции» по управленческому, бухгалтерскому и налоговому учету.

 

По табличной части «Материалы» для материалов, по которым ведется оперативный учет остатков НЗП, производится списание регистра «Материалы в производстве».

 

По табличной части «Получатели» для материалов, по которым ведется оперативный учет остатков НЗП, формируется движение приход по регистру «Материалы в производстве».

 

Тип стоимости «Фиксированная»

 

По табличной части «Материалы» производится списание регистров накопления «Незавершенное производство» по управленческому, бухгалтерскому и налоговому учету. Для материалов, по которым ведется оперативный учет остатков НЗП, производится списание регистра «Материалы в производстве».

 

По табличной части «Получатели» формируются движения приход по регистрам накопления «Незавершенное производство» по управленческому, бухгалтерскому и налоговому учету. Для материалов, по которым ведется оперативный учет остатков НЗП, формируется движение приход по регистру «Материалы в производстве».
