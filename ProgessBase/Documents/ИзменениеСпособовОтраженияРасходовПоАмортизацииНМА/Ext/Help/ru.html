Документ «Изменение способа отражения расходов по амортизации НМА»

Документ позволяет изменить способ отражения расходов по начисленной амортизации по объектам НМА и положительным результатам НИОКР.

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистров сведений, хранящих информацию об основных средствах («Способы отражения расходов по амортизации НМА (бухгалтерский учет)» и «Способы отражения расходов по амортизации НМА (налоговый учет)»).

Влияет на распределение начисленной амортизации по бухгалтерскому и налоговому учету.
