Документ «Услуги собственным подразделениям»

Документ предназначен для отражения факта оказания услуг (выполнения работ) для собственного потребления.

Документ используется только в режиме расширенной аналитики учета затрат

В документе указывается, какую услугу и каком объеме подразделение-исполнитель оказало подразделениям-получателям.

Услуга (работа) в этом документе – позиция справочника «Номенклатура».

Закладка «Исполнитель»

Указываются сведения о подразделении, которое оказало услугу:

  • Производственное подразделение, которое оказало услугу.
  • Подразделение организации
  • Номенклатурная группа, по которой собраны затраты на оказание услуги
  • Счет (БУ), Счет (НУ) - счета бухгалтерского и налогового учета, с которых будут списаны затраты (счета кредита проводок)

Подразделение, номенклатурная группа и счета могут быть заполнены автоматически по соответствующим данным документа «Установка параметров учета номенклатуры». Подразделение организации может быть заполнено автоматически - по подразделению.

Закладка «Получатели»

Указывается направление списания стоимости услуг:

  • Подразделение, которое является получателем услуг
  • Подразделение организации
  • Количество полученных услуг
  • Счета и их аналитика, на которые будут отнесены расходы:
    • Счет (БУ), Счет (НУ) - счета дебета проводок по бухгалтерскому и налоговому учету
    • Статья затрат
    • Номенклатурная группа

Подразделение организации может быть заполнено автоматически - по подразделению.

Статья затрат и номенклатурная группа также могут быть заполнены автоматически - по реквизитам услуги «Статья затрат» и «Номенклатурная группа затрат» соответственно (см. справочник «Номенклатура»).

Количество полученных услуг используется  при расчете себестоимости для распределения сумм фактической себестоимости услуг по направлениям списания.

В течение месяца услуги собственного потребления могут быть отражены в учете по плановой стоимости. Для этого необходимо установить:

  • порядок формирования учетных цен «по плановой себестоимости» (в настройках учетной политики по управленческому и/или регламентированному учету)
  • плановую себестоимость услуги
