Документ «Изменение специального коэффициента для расчета амортизации НМА»

Данный документ предназначен для изменения специального коэффициента для начисления амортизации согласно п. 10 статьи 259  НК РФ.

Влияет на начисление амортизации по объектам НМА в налоговом учете.

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующего регистра сведений «Специальный коэффициент для амортизации НМА (налоговый учет)».
