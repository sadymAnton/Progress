Документ "Расчеты по страховым взносам"

Документом регистрируются перечисленные по принадлежности суммы страховых взносов во внебюджетные фонды (ПФР, ФОМС и ФСС), включая страховые взносы на обязательное страхование от несчастных случаев.

Особенности автоматического заполнения документа

Автоматически документ заполняется только по операциям Уплата ... - остатками неуплаченных сумм по данным учета.

Ввод остатков по расчетам с фондами при начале эксплуатации

Используя операцию Начисление, можно ввести остатки задолженности при начале эксплуатации программы.
