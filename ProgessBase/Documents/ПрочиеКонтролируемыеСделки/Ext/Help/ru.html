Документ «Прочие контролируемые сделки»

Документ предназначен для регистрации произвольных контролируемых сделок, например, для операций, которые отражаются вручную документом "Операции (бухгалтерский и налоговый учет)".

 

При проведении документа формируются записи в регистр накопления "Контролируемые сделки организаций". При заполнении уведомления эти сделки влияют на формирование суммового критерия и включаются в уведомление наравне со сделками, которые зарегистрированы автоматически по данным информационной базы.

В шапке документа можно выбрать следующие реквизиты:

  • Уведомление. Уведомление о контролируемых сделках, для которого регистрируются прочие контролируемые сделки. Реквизит является обязательным для заполнения.
  • Контрагент. Контрагент, для которого регистрируются прочие контролируемые сделки. Значение реквизита выбирается из справочника «Контрагенты». Реквизит является обязательным для заполнения.

В табличной части "Сделки" указывается список контролируемых сделок. Каждая строка представляет собой отдельную сделку.

При выборе договора в валюте появляется возможность указать сумму сделки без НДС и сумму НДС в валюте договора.

Для табличной части "Сделки" существует возможность зарегистрировать внешнюю обработку заполнения табличных частей.
