Документ «Учет положений переходного периода по НДС»

Документ предназначен для учета положений переходного периода, отраженных в статье 3 федерального закона №119-ФЗ, по расходам, связанным со строительством.

В документе предусмотрено два режима заполнения:

  • при использовании документа в течение 2006 года: в соответствии с п.2 статьи 3 федерального закона N 119-ФЗ, НДС, предъявленный подрядными организациями  в течение 2005 года, по объектам, которые не приняты к учету в качестве основных средств, подлежат вычетам в течение 2006 года равными долями по налоговым периодам;
  • при использовании документа 31 декабря 2005 года: в соответствии с п.6 статьи 3 федерального закона N 119-ФЗ, НДС, вычетам подлежат суммы налога, предъявленные налогоплательщику и уплаченные им по товарам (работам, услугам), использованным при строительстве в течение 2005 года, по объектам, которые не приняты к учету в качестве основных средств.

По счетам-фактурам, перечисленным в табличной части документа, снимается блокировка с вычета НДС, связанная с отнесением НДС на строительство объектов основных средств.

Документ не формирует проводок.
