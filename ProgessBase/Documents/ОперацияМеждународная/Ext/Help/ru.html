Документ «Операция международная»

Документ «Операция международная» предназначен для ввода записей в регистр международного учета вручную.

После указания даты и выбора организации в операции вводятся одна или несколько записей (проводок), которые при проведении документа отражаются в международном учете (регистр бухгалтерии «Международный»).

В целом порядок заполнения аналогичен операции по бухгалтерскому учету, только счета и аналитика выбираются в соответствии с планом счетов международного учета.

Будьте внимательны при отражении вручную операций, связанных со счетами, где основной учет ведется с использованием специализированных регистров: это, как правило, счета учета запасов (включая затраты на производство).

В частности:

  • для учета материалов, готовой продукции, полуфабрикатов, товаров используется регистр накопления «Партии товаров на складах (международный учет)»;
  • для учета товаров отгруженных - регистр «Партии товаров переданные (международный учет)»;
  • для учета прямых затрат на производство - регистр «Незавершенное производство (международный учет)»;
  • для учета общепроизводственных расходов - регистр «Затраты (международный учет)».

Чтобы в дальнейшем информация в соответствующих регистрах не разошлась с данными счетов учета (а это абсолютно необходимо для правильной работы таких документов, как «Списание МПЗ (международный)» и «Расчет себестоимости»), нужно наряду с ручными проводками внести корректировочные записи в соответствующие регистры с помощью документа «Корректировка записей регистров накопления».
