Документ «Ввод начальных остатков по взаиморасчетам»

Документ предназначен для ввода начальных остатков по взаиморасчетам с контрагентами.

В документе предусмотрены следующие виды операций:

  • расчеты с поставщиками;
  • расчеты с покупателями;
  • расчеты по прочим операциям;
  • резервы о сомнительным долгам.

Расчеты с поставщиками

Вид операции «Расчеты с поставщиками» выбирается, если необходимо ввести остатки по расчетам с поставщиками в управленческом и бухгалтерском учете.

Остатки по расчетам и по авансам поставщикам вводятся на отдельных закладках.

Для того чтобы сформировать проводки по вводу начальных остатков в корреспонденции со счетом 000 «Вспомогательный счет», необходимо установить флажок «Формировать проводки». Если организация является плательщиком налога на прибыль, начиная с 31.12.2007, при вводе остатков по авансам в иностранной валюте будут также формироваться проводки по забалансовому счету ВАЛ.60.

Если в документе установлен флажок «Отразить расчеты для целей НДС», при проведении документа будут сделаны движения в регистр накопления «НДС расчеты с поставщиками».

Документ также можно использовать для корректировки регистра накопления «Расчеты по приобретению (бухгалтерский учет)». Для того, чтобы документ выполнил только корректировку регистра «Расчеты по приобретению (бухгалтерский учет)», необходимо установить флажки «Отражать в бух. учете» и «Отразить в регистрах расчетов организации».  Остальные флажки, в том числе флажок «Отразить в регистре взаиморасчетов по документам», в этом случае должны быть сняты.

Расчеты с покупателями

Вид операции «Расчеты с покупателями» выбирается, если необходимо ввести остатки по расчетам с покупателями в управленческом и бухгалтерском учете.

Остатки по расчетам и по авансам, полученным от покупателей, вводятся на отдельных закладках.

Для того чтобы сформировать проводки по вводу начальных остатков в корреспонденции со счетом 000 «Вспомогательный счет», необходимо установить флажок «Формировать проводки». Если организация является плательщиком налога на прибыль, начиная с 31.12.2007, при вводе остатков по авансам в иностранной валюте будут также формироваться проводки по забалансовому счету ВАЛ.62.

Если в документе установлен флажок «Отразить расчеты с контрагентами для целей НДС», при проведении документа будут сделаны движения в регистр накопления «НДС расчеты с покупателями».

Документ также можно использовать для корректировки регистра накопления «Расчеты по реализации (бухгалтерский учет)». Для того, чтобы документ выполнил только корректировку регистра «Расчеты по реализации (бухгалтерский учет)», необходимо установить флажки «Отражать в бух. учете» и «Отразить в регистрах расчетов организации».  Остальные флажки, в том числе флажок «Отразить в регистре взаиморасчетов по документам», в этом случае должны быть сняты.

Расчеты по прочим операциям

Вид операции «Расчеты по прочим операциям» выбирается, если необходимо ввести остатки по расчетам с контрагентами по прочим операциям (по договорам с видом договора «Прочее»).

На закладке «Увеличение долга» вводятся остатки, увеличивающие долг контрагента, на закладке «Уменьшение долга» —остатки, уменьшающие долг контрагента.

Для того чтобы сформировать проводки по вводу начальных остатков в корреспонденции со счетом 000 «Вспомогательный счет», необходимо установить флажок «Формировать проводки».

Резервы по сомнительным долгам

Вид операции «Резервы по сомнительным долгам» выбирается, если необходимо ввести остатки по резервам по сомнительным долгам контрагентов-дебиторов.

Для ввода резервов по сомнительным долгам в учетной политике (бухгалтерский и налоговый учет) должны быть установлены соответствующие настройки.

 

Если начальные остатки вводятся датой, начиная с 01.01.2007, и флаг «Отражать в бух. учете» установлен, при вводе остатков взаиморасчетов в иностранной валюте и в условных единицах в таблице «Расчеты» необходимо в колонке «Сумма (регл) с учетом переоценки» указывать рублевый остаток, соответствующий остатку с учетом всех проведенных до даты ввода остатков переоценок (остаток по курсу на дату ввода остатков), а в колонке «Сумма (регл)» указывать сумму рублевого остатка, определенного по курсу на дату операции (приобретения, реализации, аванса) - без учета последующих переоценок. При вводе остатков авансов в иностранной валюте, начиная с 31.12.2007, в таком же порядке заполняется таблица «Авансы».
По рублевым расчетам суммы в обеих колонках должны совпадать.

Автоматическое заполнение табличных частей по данным учета НДС

Данный сервис предназначен для перехода на ведение расчетов с контрагентами на регистрах накопления, используется для организаций на общей системе налогообложения.

Сервис предусмотрен для видов операций «Расчеты с поставщиками» и «Расчеты с покупателями», вызывается с помощью меню «Заполнить - Заполнить по данным учета НДС».

Заполнение табличных частей происходит на основании остатков регистров «НДС Расчеты с поставщиками» и «НДС Расчеты с покупателями» на дату документа.

Если расчеты по договору ведутся с детализацией по заказам или по счетам, и в договоре не установлен признак «По документам расчетов с контрагентами», сведения о сделках заполняются на основании остатков регистра «Взаиморасчеты с контрагентами».

Если расчеты по договору ведутся с детализацией по заказам или по счетам, и в договоре установлен признак «По документам расчетов с контрагентами», заполнение табличной части происходит на основании остатков регистра «Взаиморасчеты с контрагентами по документам расчетов». Данные регистров «НДС Расчеты с поставщиками» и «НДС Расчеты с покупателями» в этом случае не учитываются.

Счета учета расчетов с контрагентами заполняются на основании остатков на счетах бухгалтерского учета.

При заполнении учитываются только договоры в валюте регламентированного учета

Подробная методика перехода изложена в файле «Методика перехода на ведение расчетов с контрагентами на регистрах накопления».

Автоматическое заполнение табличных частей по данным регистра «Взаиморасчеты УСН»

Данный сервис предназначен для перехода на ведение расчетов с контрагентами на регистрах накопления, используется для организаций на УСН.

Сервис предусмотрен для видов операций «Расчеты с поставщиками» и «Расчеты с покупателями», вызывается с помощью меню «Заполнить - Заполнить по взаиморасчетам УСН».

Заполнение табличных частей происходит на основании остатков регистра «Взаиморасчеты УСН» на дату документа. Счета учета расчетов с контрагентами заполняются на основании остатков на счетах бухгалтерского учета.

Если остаток взаиморасчетов по договору на счетах бухгалтерского учета превышает остаток в регистре «Взаиморасчеты УСН», добавляется строка на сумму расхождения с незаполненными реквизитами «Документ» и «Сделка».

При заполнении учитываются только договоры в валюте регламентированного учета.

Подробная методика перехода изложена в файле «Методика перехода на ведение расчетов с контрагентами на регистрах накопления».
