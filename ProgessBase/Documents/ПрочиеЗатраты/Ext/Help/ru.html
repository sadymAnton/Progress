Документ «Прочие затраты»

Документ предназначен для отражения в расходах или списания с расходов нематериальных затрат, стоимость которых известна по реквизитам первичного документа и не подлежит корректировке при закрытии месяца. Данным документом можно отразить расходы следующего характера:

  • Производственные;
  • Брак в производстве;
  • Общепроизводственные (ОПР);
  • Общехозяйственные (ОХР);
  • Коммерческие (КР);
  • Вложения во внеоборотные активы;
  • Прочие затраты.

Документ имеет два вида операции – «отражение» и «списание». При отражении затрат затраты увеличиваются, в бухгалтерском учете такая операция соответствует списанию на затраты с кредита непроизводственных счетов. При списании затраты уменьшаются, в бухгалтерском учете такая операция соответствует списанию со счетов учета затрат в дебет непроизводственных счетов.

 

Реквизиты шапки документа:

  • Подразделение. Подразделение, на которое относятся или с которого списываются затраты по управленческому учету.
  • Подразделение организации. Подразделение, на которое относятся или с которого списываются затраты по регламентированному учету.

 Документ содержит три закладки:

  • Затраты. На этой закладке вводится вся аналитика по учету затрат и сумма затрат:

    • Статья затрат. Указывается нематериальная статья затрат, это обязательная аналитика затрат. После выбора статьи затрат в реквизите «Хар-р затрат» справочно будет показан характер затрат из выбранной статьи затрат.

    • Заказ. Заказ покупателя или заказ на производство. Распределение косвенных затрат (ОПР, ОХР, КР) если для них задан заказ, выполняется только на выпуск продукции по заданному заказу.

    • Счет затрат. Счета затрат для бухгалтерского учета;

    • Счет затрат (НУ). Счета затрат для налогового учета;

    • Аналитика. Аналитика затрат. Вид аналитики зависит от выбранного счета учета затрат.

    • Сумма. Сумма затрат по управленческому учету.

    • Сумма (БУ). Сумма затрат по бухгалтерскому учету.

    • Сумма (НУ). Сумма затрат по налоговому учету.

    Предусмотрено заполнение табличной части по кнопке «Заполнить». При нажатии кнопки «Заполнить по остаткам» открывается список выбора характера затрат. После выбора в табличную часть документа будут подобраны остатки затрат по выбранному характеру затрат. При необходимости в списке выбора можно указать вариант "<Все>" - в этом случае заполнение будет выполнено без отбора по характеру затрат.

  • Бухгалтерский учет. Закладка становится видимой при указании принадлежности документа к бухгалтерскому учету. На закладке указывает непроизводственный счет и соответствующие ему значения субконто. Это счет, с которого будет списана и отнесена на затраты сумма по бухгалтерскому учету, если вид операции в документе «отражение»; если вид операции «списание», то в дебет этого счета будут отнесены затраты с кредита счета учета затрат.
  • Налоговый учет. Закладка становится видимой при указании принадлежности документа к налоговому учету. На закладке указывает непроизводственный счет и соответствующие ему значения субконто. Это счет, с которого будет списана и отнесена на затраты сумма по налоговому учету, если вид операции в документе «отражение»; если вид операции «списание», то в дебет этого счета будут отнесены затраты с кредита счета учета затрат.
