Документ «Установка параметров учета номенклатуры»

Назначение документа:

  • Установка счетов учета номенклатуры, которые будут использоваться при проведении документов (подробнее см. ниже)
  • Установка параметров учета затрат на оказание услуг собственным подразделениям и внешним заказчикам
  • Установка способа суммовой оценки в печатных формах

Все настройки параметров учета должны быть выполнены в одном документе. Если с определенной даты требуется изменить какие-либо параметры учета, следует создать новый документ «Установка параметров учета номенклатуры» и заполнить все табличные части.

Установка счетов учета номенклатуры

С даты первого проведенного документа «Установка параметров учета номенклатуры» будет действовать режим определения счетов при проведении документов.

В данном режиме счета учета номенклатуры и затрат будут определяться при проведении документов и их заполнение в документах не обязательно. Автоматически счета учета в строках документов заполняться не будут.

Счета учета номенклатуры настраиваются на закладках «Товары» и «Услуги» документа.
Счета учета затрат определяются по виду подразделения и характеру статьи затрат.

На закладке «Документы» может быть включено использование сервиса заполнения счетов в документах (флаг «Использовать параметры учета при заполнении реквизитов документов»). Данный сервис предназначен для контроля полноты и правильности определения счетов в начальный период использования режима: счета учета в строках документов будут заполняться автоматически при заполнении документов по данным документа «Установка параметров учета номенклатуры».

Если счета учета в строках документов заполнены (вручную, с использованием сервиса заполнения или иным способом), то при проведении будут использованы указанные в документах счета и значения субконто.

Счета учета могут устанавливаться для каждого вида номенклатуры или элемента номенклатуры. Для разных организаций по одной и той же номенклатуре могут быть установлены разные счета учета. При необходимости можно устанавливать счета сразу для всех организаций и/или для всей номенклатуры. Если для реквизита «Номенклатура» указано значение «Для всей номенклатуры», то настройка будет действовать для всей номенклатуры, по которой не установлены иные счета учета.

Счета определяются с учетом степени детализации установленных реквизитов (перечислены по убыванию значимости):

  • указан элемент номенклатуры
  • указан вид номенклатуры
  • указана организация

Порядок строк в документе «Установка параметров учета номенклатуры» значения не имеет.

Например. В документе «Установка параметров учета номенклатуры» счета учета настроены двумя строками для следующих значений реквизитов:

1. «Организация» - для всех организаций, «Номенклатура» - по виду номенклатуры «Товар»

2. «Организация» - Торговый дом, «Номенклатура» - для всей номенклатуры

В организацию «Торговый дом» для номенклатуры с видом «Товар» счета учета будут определяться в соответствии с настройками первой строки.

Настройки выполняются как для товаров, так и для услуг, на соответствующих закладках документа.

Закладка «Товары»

На данной закладке настраиваются:

  • счета учета запасов на складе
  • счета учета доходов от реализации
  • счета учета расходов
  • признак принадлежности номенклатуры (при использовании Расширенной аналитики учета затрат).

В зависимости от настроенного счета учета запасов на складе определяются счета учета

  • отгруженных собственных товаров (определяется как субсчет счета 45, соответствующий счету учета запасов)
  • ТМЦ, переданных в эксплуатацию (определяется как 10.11, если счет учета номенклатуры - 10.10, и 10.09 в остальных случаях)
  • НДС
  • розницы (определяется как 41.02, если счет учета номенклатуры - 41.01)

Аналитика доходов и расходов (номенклатурная группа) указывается в справочнике «Номенклатура» (реквизит «Номенклатурная группа»).

Для принятых ТМЦ допустимо при настройке счета учета запасов указать забалансовый счет, который будет использоваться во всех операциях. Если указан балансовый счет, то для учета принятых ТМЦ используются предопределенные счета учета

  • в операциях, связанных с поступлением, использованием или выбытием материалов, полученных в переработку
    • 003.01 Материалы на складе
    • 003.02 Материалы, переданные в производство
  • в остальных операциях
    • 004.01 Товары на складе
    • 004.02 Товары, переданные на комиссию

Закладка «Услуги»

На данной закладке настраиваются:

  • Счета учета доходов от реализации услуг и связанных с ней расходов
  • Счета и аналитика учета производственных затрат, связанных с оказанием услуг
    • покупателям и заказчикам
    • собственным подразделениям
  • Признак использования учета затрат по заказам.
    Установка флага позволяет рассчитывать себестоимость оказанных услуг в разрезе заказов.

Указанные данные используются:
  • при проведении документа «Расчет себестоимости» с выбранным действием «Расчет себестоимости услуг»
  • при заполнении документа «Услуги собственным подразделениям»

Установка способа суммовой оценки в печатных формах

Позволяет определить способ формирования себестоимости в печатных формах и при заполнении документов.
Возможны следующие варианты:

  • По данным управленческого учета
  • По данным бухгалтерского учета
  • По типу цен (любой тип цен)

По умолчанию используется вариант: «По данным бухгалтерского учета».

Для вариантов «По данным управленческого учета» и «По данным бухгалтерского учета»:

  • Если используется расширенная аналитика учета затрат то себестоимость определяется в среднем за месяц
  • Если расширенная аналитика не используется, то себестоимость определяется на дату документа.
