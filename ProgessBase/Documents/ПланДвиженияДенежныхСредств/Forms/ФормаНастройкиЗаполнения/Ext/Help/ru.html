"Настройка автоматического заполнения"

В форме выполняется настройка правил заполнения документа "План движения денежных средств" для заданных источников планирования.

Форма настройки имеет два экрана (переключение между которыми производится при помощи кнопки "Перейти к поступлениям/Перейти к платежам"):

  • Планирование поступлений денежных средств
  • Планирование платежей

Каждый экран содержит источники планирования, расположенные на нескольких закладках. На первом экране настраиваются правила заполнения планируемых поступлений денежных средств, на втором - правила заполнения планируемых платежей.

Все закладки настраиваются по общему принципу. На одной закладке располагаются одновременно два источника, относящиеся к одному и тому же разделу учета. Слева используются фактические данные за выбранный период, справа - плановые данные. Исключение - источник "Задолженность по заработной плате", где рассматриваются только фактические данные. Выбор источника зависит от целей пользователя.

Источники планирования "Планы поступлений денежных средств" (закладка "Поступления денежных средств" экрана "Планирование поступлений денежных средств") и "Планы платежей" (закладка "Платежи за период" экрана "Планирование платежей") рассчитываются как планы поступления и списания денежных средств из ранее проведенных документов "План движения денежных средств".

Обязательным для каждого выбранного источника является заполнение реквизита "Процент от суммы", который определяет отношение планируемой суммы к сумме источника.

Для каждого источника можно задать дополнительную аналитику (реквизиты "Установить статью/Статья движения денежных средств" и "Установить проект/Проект").

При необходимости можно настроить автоматическое заполнение в документе измерений "Контрагент", "Договор контрагента" и "Сделка", установив соответствующие флажки в группе полей "Детализация".

Закладка "Отборы" каждого экрана используется для фильтрации выбранных источников. На ней можно устанавливать ограничения на отбираемые для заполнения документа источники. Закладку нужно заполнять отдельно для каждого из экранов.

 

После выполнения всех настроек можно заполнить документ (кнопка "Выполнить").

Предварительные прогнозы движения денежных средств можно проанализировать при помощи отчета "Планы движения денежных средств".

Запланированные движения денежных средств можно сравнить с фактическими с помощью отчета "Сравнительный анализ движения денежных средств".
