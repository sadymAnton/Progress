Документ "Право на дополнительные дни отпуска"

Документ предназначен для ввода дополнительных дней управленческого отпуска, обычно называемых "отгулами".

 

Документ позволяет ввести дополнительные дни управленческого отпуска списку сотрудников. Дни отпуска записываются на дату документа. По каждому отпуску можно ввести примечание. Это примечание будет выводиться в справке по отпуску, которую можно открыть из обработки "График отпусков".
