Документ «Реализация отгруженных товаров»

Документ «Реализация отгруженных товаров» предназначен для фиксации факта реализации  товаров ранее отгруженных без перехода права собственности документом «Реализация товаров и услуг» с видом операции «отгрузка без перехода права собственности».

Реализация производится на основании документа отгрузки, указываемого в реквизите «Документ отгрузки».

Информация о номенклатурном составе, количистве, цене и счетах учета товаров берется из указанного документа отгрузки.

Внимание: Отгрузка без перехода права собственности доступна только при использовании режима расширенной аналитики учета затрат.
Не поддерживается отгрузка без перехода права собственности для договоров, по которым ведутся взаиморасчеты по документам расчетов с контрагентами.
