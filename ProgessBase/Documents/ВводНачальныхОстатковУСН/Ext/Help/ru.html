Документ «Ввод начальных остатков УСН»

Документ предназначен для ввода остатков по налоговому учету в организациях, применяющих упрощенную систему налогообложения с объектом налогообложения «Доходы, уменьшенные на величину расходов». Так же, документ позволяет скорректировать первоначальные ведения об объектах основных средств и нематериальных активов, используемые в налоговом учете по упрощенной системе налогообложения.

Данный документ формирует движения по регистрам накопления, используемым в учете по УСН при признании расходов.

«Взаиморасчеты»

Закладка «Взаиморасчеты» предназначена для ввода информации о состоянии взаиморасчетов. Закладка содержит подзакладки с разными видами взаиморасчетов, состав колонок в табличной части каждой подзакладки зависит от степени детализации, необходимой для данного вида взаиморасчетов. Список подзакладок закладки «Взаиморасчеты»:

  • С покупателями
  • С поставщиками
  • С подотчетниками
  • С сотрудниками
  • По налогам

«Партии товаров на складах»

Закладка «Партии товаров на складах» предназначена для ввода информации о партиях ТМЦ, находящихся в организации.

«Партии товаров переданные»

Закладка «Партии товаров на складах» предназначена для ввода информации о партиях ТМЦ, переданных на комиссию или в переработку.

«Расходы УСН»

Закладка «Расходы УСН» предназначена для ввода информации об остатках расходов, возникших у организации, но еще не признанных уменьшающими налоговую базу (не оплаченных поставщику, не переданных в производство и прочее). Закладка содержит подзакладки с разными видами расходов, состав колонок в табличной части каждой подзакладки зависит от степени детализации, необходимой для данного вида расходов. Список подзакладок закладки «Взаиморасчеты»:

  • Номенклатура
  • Нематериальные активы (не используется с 2006 года)
  • Услуги
  • РБП
  • Расходы на оплату труда
  • Налоги и взносы

«Корректировка сведений»

Закладка «Корректировка сведений» предназначена для установки и изменения первоначальных сведений налогового учета УСН по основным средствам и нематериальным активам, находящимся в эксплуатации. Список подзакладок закладки «Корректировка сведений»:

  • Основные средства
  • Нематериальные активы
