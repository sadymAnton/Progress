Отчет "Контактная информация"

Отчет предназначен для отображения контактной информации  контрагентов. Также в отчете можно посмотреть контактную информацию по физическим и юридическим лицам контрагентов, по всем организациям и сотрудникам (физическим лицам). 

Пользователь в настройках отчета должен указать объект отчета:

  • Контрагенты;
  • Физические лица;
  • Организации.


В настройках отчета флажками нужно отметить виды контактной информации, которые необходимо отображать, или строку — для персональных значений видов контактной информации.

Список видов контактной информации может быть заполнен путем выбора нужных элементов из справочника Виды контактной информации. Также новый вид контактной информации может быть введен непосредственно в процессе заполнения базы данных — в этом случае он будет записан в регистр сведений.

Для каждого объекта отчета используется свой список видов контактной информации и соответствующие группировки, которые настраиваются в диалоговом окне настройки отчета, вызываемом по нажатию на кнопку Настройка.
