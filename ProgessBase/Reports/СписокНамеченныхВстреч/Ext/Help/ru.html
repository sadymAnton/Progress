Отчет "Список намеченных встреч с кандидатами"

Отчет выводит информацию обо всех запланированных встречах сотрудников нашего предприятия с кандидатами за выбранный период.

Отчет по умолчанию настроен так, чтобы выводить список встреч за текущий месяц для каждого сотрудника с группировкой по подразделениям.

Возможные показатели отчета

  • Начало встречи
  • Конец встречи

Возможные группировки отчета

  • Подразделение и должность сотрудника
  • Сотрудник
  • Документ
