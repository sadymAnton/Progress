Отчет "Анализ налогового учета по УСН"

Отчет предназначен для анализа структуры доходов и расходов налогового учета по УСН.

Отчет состоит общей из схемы налоговой базы и расшифровок отдельных элементов этой схемы.

Для заполнения схемы данными и получения доступа к расшифровкам конкретных видов доходов и расходов необходимо:

  • указать период (начальную и конечную даты отчета);
  • указать организацию;
  • нажать кнопку "Сформировать";

Отчеты по видам доходов и расходов

Отчеты по видам доходов и расходов доступны по двойному клику мыши на блоке с интересующими показателями.

Для перехода между отчетами разного уровня используются кнопки "Назад" и "Структура налоговой базы".
