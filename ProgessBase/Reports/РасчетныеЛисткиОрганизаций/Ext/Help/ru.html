Отчет "Расчетные листки организаций"

Отчет предоставляет возможность сформировать расчетные листки сотрудников организаций за произвольный период времени, отобранные и сгруппированные по различным критериям.

Отчет по умолчанию настроен так, чтобы в результате получились расчетные листки за текущий месяц, сгруппированные по организациям и их подразделениям.

Возможные группировки отчета

  • Организация
  • Подразделение и Должность
  • Сотрудник
  • Месяц начисления

Дополнительные возможности

Отчет предоставляет возможность формировать краткие расчетные листки (предлагается по умолчанию) для выдачи на руки сотрудникам и подробные листки для проведения анализа результатов расчетов, в которых помимо сумм начислений и оплаченного времени выводятся периоды действия начислений и исходные показатели для расчета (тарифные ставки, проценты оплаты и т.п.).
