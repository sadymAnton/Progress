Отчет "Оценка эффективности работы ОУП"

Отчет предоставляет возможность получить показатели эффективности набора персонала

Показатели отчета:

  • Коэффициент набора персонала (отношение количество заявок кандидатов к количеству вакансий)
  • Количество обслуживаемых сотрудников
  • Количество обслуживаемых юридических лиц
  • Количество обученных сотрудников
  • Количество проведенных обучений
  • Качество отбора (отношение количества прошедших испытательный срок к общему количеству принятых сотрудников)
  • Средняя продолжительность закрытия вакансий

    Разрезы, в которых можно выводить выше указанные показатели:

  • Год, квартал, месяц, неделя, дата
  • Ответственное лицо (только для показателей: коэффициент набора персонала, качество отбора, средняя продолжительность закрытия вакансий, количество обученных сотрудников, количество обслуживаемых сотрудников, количество обслуживаемых юридических лиц)
  • Подразделение, должность (только для показателей: коэффициент текучести кадров, количество обученных сотрудников)
