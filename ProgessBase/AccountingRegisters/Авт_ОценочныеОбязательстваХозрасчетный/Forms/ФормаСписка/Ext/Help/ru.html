Журнал проводок

Журнал проводок является основной формой списка регистра бухгалтерии «Бухгалтерский учет» и представляет собой удобный инструмент для просмотра, отбора, поиска и редактирования проводок. Из этого журнала отобранные проводки могут быть стандартным образом распечатаны.

Реквизиты отбора «Счет» и «Организация» вынесены в форму журнала проводок исключительно с целью упрощения этих отборов и не имеют никакого преимущества перед аналогичными реквизитами стандартного диалога отбора и сортировки этой формы. Более того, они однозначно связаны с ними.

По двойному щелчку мыши на проводке возможен переход к документу, сформировавшему эту проводку.

Из журнала возможен вызов функции проверки проводок (кнопка командной панели «Проверка проводок»). Проверка осуществляется за тот период, который установлен для журнала на момент начала проверки. В процессе проверки корреспонденции всех проводок сравниваются со списком корректных проводок, хранящихся в регистре сведений «Корректные проводки». При обнаружении некорректных корреспонденций выдается соответствующее сообщение.
