Настройки

Форма «Настройки» предназначена для управления свойствами формы «Регламентированная и финансовая отчетность» и общими свойствами регламентированных отчетов.

Форма вызывается нажатием кнопки «Настройка» в верхней командной панели формы «Регламентированная и финансовая отчетность».

«Проверять соотношения при печати из журнала» - если флажок установлен, то при печати отчета, будет запускаться механизм проверки соотношений показателей, для тех отчетов, где предусмотрен контроль показателей. В случае, если будут найдены ошибки, то появится информационное окно, с названием отчета и сообщением о необходимости проверки показателей.

«Показывать «пустые» документы в журнале выгрузки» – если флажок установлен, в журнале выгрузки будут отображаться документы «Выгрузка регламентированных отчетов», в которых не выгружалось ни одного отчета.

«Фильтровать дерево видов отчетов при установленном отборе по периоду» – список видов отчетов в форме «Регламентированная и финансовая отчетность» имеет возможность динамической фильтрации в зависимости от отчетного периода.

Если такой режим показа включен, то при установленном отборе по периоду в списке видов отчетов будут показываться только те отчеты, которые представляются за выбранный отчетный период.

Настройки календаря

«Не отображать ближайшие события» – если флажок установлен, то на форме календаря бухгалтера не будут отображаться события, которые должны наступить, но по состоянию на выбранный день в календаре ещё не наступили.

«Запоминать выбранный период» – если флажок установлен, то календарь бухгалтера при очередном открытии будет устанавливать тот период, который был выбран в прошлом сеансе работы с календарем.

«Напоминать о событиях ежедневно за … дня(ей) до наступления» – количество дней до наступления выбранного в календаре бухгалтера события. Напоминание о событиях будут выдаваться ежедневно.

Печать машиночитаемых форм

«Для установки компоненты печати машиночитаемых форм нажмите здесь» – по нажатию ссылки «здесь» будет запущена программа установки универсальной компоненты печати «Печать НД с PDF417» разработки ФНС России. Дистрибутив компоненты входит в состав конфигурации, и, если компонента не установлена на компьютере пользователя, она устанавливается автоматически при первой попытке напечатать какой-либо регламентированный отчет в виде машиночитаемого бланка с двухмерным штрихкодом стандарта PDF417. При необходимости, компоненту можно переустановить принудительно,  для этого используется данный управляющий элемент формы «Настройки».

«Для настройки шаблонов печати машиночитаемых форм нажмите здесь» – по нажатию ссылки «здесь» вызывается форма для загрузки щаблонов машиночитаемых форм регламентированой отчетности. Шаблоны машиночитаемых форм включены в состав конфигурации и устанавливаются вместе с ней, однако, ФНС России периодически обновляет шаблоны, поэтому может возникнуть необходимость обновить шаблоны без обновления самой конфигурации. Для этого служит форма загрузки шаблонов, для вызова которой используется данный управляющий элемент.

Настройки форм отчетов статистики

«Для настройки XML-шаблонов ЭВФ отчетов статистики нажмите здесь» – по нажатию ссылки «здесь» вызывается форма для загрузки XML-шаблонов электронных версий форм отчетов статистики. XML-шаблоны содержат описание структуры и правил формирования отчетов статистики, информацию, которая должна быть в отчете для однозначной идентификации формы при приеме в ЕССО, а также - описание элементов визуализации и контроля правильности заполнения данных. Используются в отчете «Прочие формы статистической отчетности».

«Для настройки формата выгрузки отчетов статистики нажмите здесь» – по нажатию ссылки «здесь» вызывается форма настройки формата выгрузки статотчетности в электронном виде. В формате, указанном для организации, будет производиться выгрузка всех статистических отчетов. Формат выгрузки должен соответствовать версии системы сбора статотчетности, используемой территориальным органом Росстата.

Оповещения

«Для настройки оповещений нажмите здесь» – по нажатию ссылки «здесь» вызывается форма настройки параметров сервиса оповещений регламентированной отчетности.

Прочие общие настройки

«Автосохранение каждые … минут» – если флажок установлен, при редактировании регламентированного отчета он будет автоматически сохраняться с указанным интервалом времени. Для новых отчетов автоматическое сохранение включается после того, как отчет будет записан в журнале отчетов.

Документооборот с контролирующими органами

«Для настройки параметров документооборота нажмите здесь» – по нажатию ссылки «здесь» вызывается форма настройки параметров документооборота с контролирующими органами.
