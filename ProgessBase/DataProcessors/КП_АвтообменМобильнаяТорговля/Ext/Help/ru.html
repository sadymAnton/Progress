Формат строки запуска 1С для выполнения обмена с последующим завершением сеанса для клиент-серверной базы 1С:
<Путь до исполняемого файла 1С> ENTERPRISE /S"<имя кластера серверов 1С>\<имя базы в кластере>" /N"<имя пользователя>" /P"<пароль>" /Execute "<полное имя файла обработки СТ_Автообмен>" /C"Exchange" /DisableStartupMessages 

Для файловой базы 1С:
<Путь до исполняемого файла 1С> ENTERPRISE /F"<каталог информационной базы>" /N"<имя пользователя>" /P"<пароль>" /Execute "<полное имя файла обработки СТ_Автообмен>" /C"Exchange" /DisableStartupMessages

Описание параметров:

/Execute  - предназначен для того, чтобы при открытии 1С была открыта указанная обработка

/C - передача параметров в сеанс 1С. Параметр "Exchange" служит флагом для обработка автообмена, что нужно выполнить обмен и завершить сеанс 1С. 

/DisableStartupMessages - отключает возможные служебные сообщения при старте сеанса 1С

 

Пример строки запуска 1С:
"C:\Program Files (x86)\1cv81\bin\1cv8.exe" ENTERPRISE /S"V-PANOV-M\Cherk" /N"Администратор" /P"123456" /Execute "C:\Sync\СТ_Автообмен.epf" /C"Exchange" /DisableStartupMessages

 

Также возможно использование параметра /Out (не является обязательным):
/Out"<полное имя txt-файла лога>" -NoTruncate

При использовании данного параметра все сообщения, которые выводятся в окно сообщение 1С (в частности, о процессе выгрузи/загрузки данных), будут записываться в указанный в параметре текстовый файл. При отсутствии указанного файла будет создан новый.
