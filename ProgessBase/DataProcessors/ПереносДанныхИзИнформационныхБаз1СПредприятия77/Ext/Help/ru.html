Перенос данных из информационных баз 1С:Предприятия 7.7

Обработка "Перенос данных из информационных баз 1С:Предприятия 7.7" - это специализированный Помощник перехода, предназначенный для переноса данных из информационных баз на платформе 1С:Предприятие 7.7.

Помощник перехода, позволяет произвести автоматическую загрузку данных непосредственно из информационной базы на платформе 1С:Предприятие 7.7. Для этого потребуется указать из какой базы производится перенос данных, а также выбрать, за какой период необходимо переносить расчетные данные. После загрузки, Помощник производит проверку загруженных данных (например, проверяются совпадение итоговых значений свода начислений и т.д.). В случае возникновения каких-либо ошибок, Помощник позволяет сформировать отчет об ошибках с указанием этапа, на котором они возникли. Для переноса данных используются правила, которые хранятся в специальном справочнике "Конвертации из информационных баз 1С:Предприятия 7.7".

Помощник перехода позволяет также загружать данные из файла выгрузки, созданного при помощи универсальной обработки выгрузки данных. При выгрузке данных из конфигураций на платформе 1С:Предприятие 7.7 при помощи универсальной обработки, рекомендуется использовать файлы обработок и правил, которые хранятся в специальном справочнике "Конвертации из информационных баз 1С:Предприятия 7.7".
