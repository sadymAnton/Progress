Назначение схемы

Схема наглядно изображает систему прав и настроек пользователей.

Также представляет собой:

  • Рабочее место ответственного за настройку прав
  • Средство изучения системы прав

Описание схемы

Каждый элемент схемы изображает механизм системы прав и настроек пользователей.

Связи между элементами схемы изображают соотношение сущностей между собой. Например, профиль может быть назначен многим пользователям, но пользователю можно назначить только один профиль - связь "Один ко многим". Или, например, пользователь может входить в несколько групп, а группа может включать несколько пользователей - связь "Многие ко многим".

Интерактивные возможности

 

Двойной щелчок мыши на интерактивных элементах/линиях связи схемы открывает соответствующую форму.

В зависимости от настроек программы некоторые элементы схемы могут быть доступными или недоступными. Элементы «Права доступа», «Группы объектов» будут не доступны, если в информационной базе не используется ограничение на уровне записей. Элемент «Группы объектов» будет доступен, когда используется хотя бы один соответствующий вид объектов доступа (Контрагенты, Физические лица, Заявки кандидатов).

  • Профили - открывает справочник Профили полномочий пользователей
  • Дополнительные права - открывает сервис настройки дополнительных прав для групп пользователей и пользователей
  • Пользователи - открывает справочник Пользователи
  • Группы пользователей - открывает справочник Группы пользователей
  • Права доступа - открывает сервис для настройки прав доступа
  • Группы объектов - открывает сервис для настройки групп доступа
  • Линия связи "Пользователи"-"Профили" - открывает сервис "Групповая обработка пользователей".
