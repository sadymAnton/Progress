Обработка "Подготовка данных ПФР"

Обработка "Подготовка данных ПФР" предназначена для формирования отчетности в Пенсионный фонд за отчетные периоды до 2010 года.

Она позволяет автоматически или вручную заполнить данные о стаже и взносах сотрудников, а также суммы уплаченных и начисленных взносов в Пенсионный фонд. Если в конфигурации используется ограничение прав доступа для кадровика и расчетчика, то в обработке это тоже учитывается: пользователи с ролью кадровик не могут видеть данные о начисленных взносах по сотрудникам и в целом по организации, а пользователи с ролью расчетчик не могут редактировать стаж сотрудников.

При начале формирования отчетности нужно заполнить реквизиты в шапке: Организацию, год, формат файла и тип сведений. Для автозаполнения отчетности нужно нажать кнопку "Заполнить", при заполнении пенсионных данных будет запрошена дата, по которую необходимо подавать сведения, и список сотрудников, выходящих на пенсию. При ручном вводе данных нужно  в разделе "Документы, пачки документов" добавить пачку АДВ-11, а в нее добавить пачки СЗВ-4.

Для документа АДВ-11 необходимо заполнить реквизиты: Дата заполнения, номер пачки, Дата на которую составляется документ. Расчеты с ПФР заполняются в табличной части "Уплата страховых взносов на обязательное пенсионное страхование", ее можно заполнить автоматически по кнопке "Заполнить расчетами с ПФР".

В разделе "Состав пачки" показываются данные по выбранной пачке СЗВ-4: список физических лиц и данные по ним. Для каждого физического лица указываются суммы начисленных взносов по страховой и накопительной части, а также сумма по дополнительному тарифу, если он является членом летного экипажа. Если организация, по которой подготавливаются данные, имеет сложную структуру (имеется головная организация или дочерние компании или обособленные подразделения), то для сотрудников, которые в течение года работали в разных обособленных подразделениях или компаниях в рамках одной структуры, нужно указать суммы взносов по всем местам работы в разделе "Всего взносов". Если у сотрудника поменялся адрес проживания, то для него указывается "Адрес для информирования". Затем указываются данные о стаже за отчетный период в разделах "Периоды отсутствия" и "Стаж работы за отчетный период".

Полноту и правильность заполнения данных можно проверить по кнопке "Проверить". Если на компьютере установлена программа проверки файлов  "Check XML", то в выпадающем меню кнопки "Проверить" по команде "Текущую пачку в CheckXML" можно запустить проверку пачки в программе CheckXML.

Для печати документов предназначена кнопка "Печать", просмотреть формируемые файлы по пачкам можно по кнопке "Показать файл", а для формирования файлов и записи на диск предназначена кнопка "На диск". Для закрытия обработки и подтверждения записи данных предназначена кнопка "ОК".

После того, как отчетность сдана и принята в ПФР необходимо запретить редактирование принятых пачек, для этого нужно нажать на кнопку "Принято в ПФР" в командной панели пачек документов.
