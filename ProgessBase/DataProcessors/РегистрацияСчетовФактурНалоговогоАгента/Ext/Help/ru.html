Обработка "Регистрация счетов-фактур налогового агента"

Обработка предназначена для автоматического формирования счетов-фактур при исполнении обязанностей налогового агента. Регистрация счетов-фактур может выполняться за один день или за период.

По кнопке Заполнить список заполняется платежами по договорам с исполнением обязанностей налогового агента. При заполнении определяются ранее выписанные счета-фактуры, они будут обновлены при выполнении обработки.

Автоматически подставляется ставка, по которой будет рассчитан начисляемый НДС - 18/118%, она может быть изменена вручную.

Выписка новых счетов-фактур и обновление ранее зарегистрированных производится по кнопке Выполнить. При выполнении в окне сообщений выводится информация о ходе регистрации счетов-фактур.

Список счетов-фактур налогового агента с отбором за указанный период можно просмотреть по кнопке Список счетов-фактур (выд.).

Групповая печать счетов-фактур может быть выполнена с помощью универсальной обработки Групповая обработка справочников и документов.
