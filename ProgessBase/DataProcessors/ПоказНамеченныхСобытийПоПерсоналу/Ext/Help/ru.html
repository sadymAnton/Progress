Запланированные кадровые перестановки

Обработка Запланированные кадровые перестановки предназначена для подтверждения намеченных ранее событий.

Рекомендуется запускать эту обработку с определенной периодичностью, например, раз в месяц, когда уже поступили сведения о выходах сотрудников на работу, возвращении в подразделения, откуда они временно переводились и т.д.

В этой обработке нужно указать период просмотра намеченных событий и кнопкой Заполнить обновить список событий. Далее, нужно отметить те события, которые уже произошли и требуют регистрации.

Кнопка Создать сформирует необходимые документы, которые будут заполнены в соответствии с характером намеченных событий:

  • Для подтверждения увольнений будет создан документ Увольнение;
  • Для возвратившихся из отпуска или командировки будет создан документ Возврат на работу;

  • Для временно переведенных в другое подразделение (на другую должность) — очередное кадровое перемещение.
