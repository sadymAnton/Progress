Обработка "Загрузка результатов анкетирования"

Обработка предназначена для загрузки результатов анкетирования из почты и каталога. Форма обработки состоит из двух закладок Загрузка из почты и Загрузка из каталога. 

Загрузка из почты

Загрузка результатов анкетирования, пришедших по электронной почте, возможна как при использовании внутреннего, так и внешнего почтового клиента.

Важно! Для того, что бы можно было заполнить список писем, полученных с помощью внешнего почтового клиента, необходимо снять флажок Только непрочитанные входящие письма.

  • Для заполнения табличного поля закладки списком писем, в которых присутствуют анкеты, необходимо нажать на кнопку Обновить.
  • При загрузке писем из программы Microsoft Office Outlook 2003 появится диалоговое окно с предупреждением. В диалоговом окне необходимо установить флажок Разрешить доступ на, и выбрать значение из списка — 10 минут.
  • После заполнения списка необходимо расставить флажки для каждой анкеты.
  • Установка флажка Эл. справочника означает, что информация из анкеты будет загружена в справочник. Вид справочника определяется в анкете.
  • Установка флажка Опрос означает, что информация из анкеты будет загружена в документ Опрос.
  • Прежде чем загружать информацию из анкеты, можно посмотреть, какая информация хранится в ней. Для этого необходимо выбрать анкету — интересующая информация отобразится в табличном поле HTML.
  • Для загрузки информации из анкеты необходимо нажать на кнопку Загрузить.

Загрузка из каталога

  • Для заполнения списка файлов необходимо выбрать каталог, в котором находятся анкеты и нажать на кнопку Обновить.
  • После заполнения списка необходимо расставить флажки для каждой анкеты.
  • Установка флажка Эл. справочника означает, что информация из анкеты будет загружена в справочник. Вид справочника определяется в анкете.
  • Установка флажка Опрос означает, что информация из анкеты будет загружена в документ Опрос.
  • Прежде чем загружать информацию из анкеты, можно посмотреть, какая информация хранится в ней. Для этого необходимо выбрать анкету; ее содержимое отобразится в табличном поле HTML.
  • Для загрузки информации необходимо нажать на кнопку Загрузить.
