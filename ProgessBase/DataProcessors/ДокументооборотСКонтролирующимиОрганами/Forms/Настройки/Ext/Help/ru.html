Настройки защищенного документооборота

Форма «Настройки защищенного документооборота» предназначена для управления параметрами электронного документоборота с контролирующими органами (ФНС России, Пенсионным фондом России, Фондом социального страхования).

Форма вызывается нажатием ссылки «здесь» в строке «Для настройки параметров документооборота нажмите здесь» формы настроек формы «Регламентированная и финансовая отчетность».

Общие настройки

Переключатель позволяет указать, следует ли использовать модуль документоборота, встроенный в конфигурацию, или внешний модуль.

При выборе пункта «Использовать внешний модуль» следует указать расположение внешнего модуля документооборота. После нажатия кнопки «ОК» формы настроек, указанный внешний модуль будет загружен в информационную базу и использоваться вместо модуля конфигурации.

«Критопровайдер» – если на компьютере, с которого предполагается осуществлять взаимодействие с контролирующими органами по каналам связи, установлены как СКЗИ КриптоПро CSP, так и СКЗИ ViPNet CSP, необходимо указать программе, какое из указанных криптосредств следует использовать.

Так как при установке на компьютере двух СКЗИ возможны конфликты при их работе, устанавливаться два СКЗИ не рекомендуется.

Персональные настройки

«Разрешить обновление модуля через Интернет (рекомендуется)» – установка данного флажка разрешает обновление модуля электронного документооборота с сервера фирмы «1C» через Интернет. Проверка наличия обновленного модуля документооборота выполняется автоматически при открытии формы «Регламентированная и финансовая отчетность».

«Для доступа в Интернет необходима авторизация на прокси» - если для доступа в Интернет по протоколу http с компьютера, с которого предполагается осуществлять взаимодействие с контролирующими органами по каналам связи, необходима авторизация на прокси-сервере, следует установить этот флажок и задать имя пользователя и пароль в полях ниже.

Настройка задается отдельно для каждого пользователя информационной базы.

Настройка будет использоваться программой при доступе в Интернет для целей проверки наличия обновлений модуля защищенного документооборота, а также для абонентов компании «Такском» при предварительной проверке регламентированных отчетов и при автоматической настройке параметров учетной записи абонента.
