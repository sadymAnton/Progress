Алгоритмы расчетов

На закладке Алгоритмы расчетов указываются некоторые особенности алгоритмов расчетов.
 
Первая особенность касается удержания за неотработанный отпуск при увольнении. В соответствии с Трудовым кодексом РФ работодатель имеет право на удержание за неотработанные сотрудником дни отпуска (например, в том случае, если сотрудник «отгулял» отпуск авансом и увольняется). В зависимости от установки переключателя Удержание за неотработанный отпуск при увольнении удержанные суммы будут рассматриваться программой:

  • как удержание, не уменьшающее налоговые базы по НДФЛ и ЕСН,
  • как корректировку ранее сделанного начисления, уменьшающую налоговые базы по НДФЛ и ЕСН.

Вторая особенность касается пересчета месячного оклада в часовую ставку для расчета оплаты сверхурочных часов.  С помощью переключателя При пересчете месячного оклада в часовую ставку использовать можно выбрать один из следующих вариантов:

  • Месячная норма времени по графику сотрудника – при пересчете будет использоваться месячная норма времени в часах по графику сотрудника; этот порядок должен быть закреплен в коллективном договоре.
  • Среднее за год количество часов в месяце – в этом варианте при пересчете будет использоваться среднемесячное количество часов с учетом длительности рабочей недели сотрудника.
  • Месячная норма по производственному календарю - в этом варианте при пересчете будет использоваться среднемесячное количество часов с учетом длительности рабочей недели по производственному календарю в соответствии с Приказом Минздравсоцразвития России от 13.08.2009 № 588н.

Кадровый учет


На закладке Кадровый учет указываются особенности кадрового учета раздельно по Организациям.
 
Если установить флажок Количество ставок и(или) Размеры ставок из группы Контролировать по штатному расписанию при кадровых изменениях, то при вводе новых кадровых документов программа будет проверять указанные параметры на соответствие штатному расписанию.

Если установить флажок Единый нумератор кадровых документов, то будет применяться сквозная нумерация при вводе документов Прием на работу, Кадровое перемещение, Увольнение и аналогичных.

Если установить флажок Показывать табельные номера в документах, то в формах документов будет отображаться колонка с табельными номерами сотрудников.

Если установить флажок Показывать табельные номера в документах, то в формах документов будет отображаться колонка с табельными номерами сотрудников.

На закладке также имеется возможность управления результатом расчета количества дней компенсации отпуска при увольнении. Если установить флажок Округлять дни компенсации отпуска при увольнении, то округление будет проводиться до целых дней.

Переключатель «Остатки отпусков будут уменьшаться при регистрации фактических отпусков» указывает какими документами будет производится списание фактических отпусков кадровыми или расчетными.

Расчет зарплаты

На закладке Расчет зарплаты указываются особенности расчета зарплаты раздельно по Организациям.

Если установить флажок Контролировать пересечение периодов начислений, регистрирующих норму времени, то программа будет контролировать факт двойной оплаты одного и того же периода времени.

Если установить флажок При начислении НДФЛ принимать исчисленный налог к учету как удержанный, то начисленный НДФЛ при проведении документов сразу же будет регистрироваться как удержанный. Такой подход упрощает формирование отчетности в Организации, где заработная плата выплачивается регулярно и без задержек.

Используются начисления в валюте - флаг используется, если в программе предусмотрены начисления в иностранной валюте.

Флажок Расчет зарплаты по ответственным используется в ситуации, когда за расчет зарплаты по подразделениям отвечают расчетчики. Это позволяет каждому расчетчику работать только с сотрудниками, которые закреплены за ним. При установленном флажке документ начисления заплаты будет заполняться списком сотрудников, закрепленным за расчетчиком, если предварительно в поле Ответственный документа указать расчетчика.

При расчете больничных не учитывать праздники, приходящиеся на период отпуска - флаг позволяет устанавливать порядок включения праздничных дней, приходящихся на период ежегодных основных и дополнительных отпусков, в расчетный период при определении среднего заработка для выплаты пособий социального страхования: по временной нетрудоспособности, по беременности и родам и по уходу за ребенком.

Процент доплаты в ночное время - процент доплаты за работу в ночное время.

Процент доплаты в вечернее время - процент доплаты за работу в вечернее время.

Тариф почтового сбора - выбор варианта тарифа почтового сбора.

Выплата зарплаты

На закладке Выплата зарплаты указываются особенности начисления аванса по Организациям.

Если установить флажок Взаиморасчеты по зарплате ведутся в разрезе месяцев ее начисления, то программа будет учитывать задолженности по зарплате с детализацией по месяцам (в противном случае – только общей суммой).

Если установить флажок Упрощенный учет взаиморасчетов, то заработная плата будет считаться выплаченной при проведении документов «Зарплата к выплате сотрудников организаций», без создания платежных документов. 

Если установить флажок Учитывать невыход, то при формировании списка на получение планового аванса будут исключены сотрудники, у которых на расчетную дату планового аванса не было зарегистрировано отработанных дней.

Бухучет зарплаты

На закладке Бухучет зарплаты задается порядок учета расходов на переходящие отпуска с 2011 года раздельно по Организациям. Предусмотрены следующие варианты:

  • в месяце начисления - расходы признаются в бухгалтерском и налоговом учете в месяце начисления отпускных;
  • БУ в месяце начисления, НУ в месяце, на который приходится отпуск - расходы признаются в бухгалтерском учете в месяце начисления отпускных, а в налоговом учете в тех месяцах, на которые пришелся отпуск;
  • в месяце, на который приходится отпуск - расходы признаются в бухгалтерском и налоговом учете в тех месяцах, на которые пришелся отпуск. 

Оценочные обязательства

На закладке Оценочные обязательства производится настройка параметров по отражению оценочных обязательств и резервов в учете с 2011 года раздельно по Организациям.

Формировать оценочные обязательства в бухгалтерском учете - при взведенном флаге будут формироваться проводки по отражению оценочных обязательств в бухгалтерском учете.

Формировать резервы в налоговом учете по налогу на прибыль - при взведенном флаге будут формироваться проводки по отражению оценочных обязательств в налоговом учете. Включение данной настройки возможно только при выбранной настройке Формировать оценочные обязательства в бухгалтерском учете.

При включенной настройке формирования по оценочным обязательствам, необходимо указать как будет исчисляться база для расчета оценочных обязательств и резервов. Для этого необходимо выбрать один из вариантов:

  •  Суммы начислений по оплате труда и исчисленные страховые взносы;
  •  Суммы начислений по оплате труда.

В группе Оценочные обязательства и резервы, размеры отчислений на: необходимо выбрать оценочное обязательство, указать размер отчислений и задать расчетную базу для начислений.

Страховые взносы

На закладке Страховые взносы можно увидеть информацию о:

  • действующих ставках страховых взносов в ПФР,ФОМС и ФСС для различных тарифов.

    Сведения о тарифах страховых взносов выводятся справочно. Установка тарифа для каждой организации выполняется в форме «Учетная политика (бухгалтерский и налоговый учет)»;

  • размер предельной величины базы страховых взносов, а также историю изменения значения базы.

На закладке раздельно по Организациям или ее обособленным подразделениям указывается страховой тариф на обязательное страхование от несчастных случаев на производстве и профессиональных заболеваний.

Расчет страховых взносов

На этой закладке собраны различные настройки по расчету страховых взносов.

Используется труд членов летных экипажей - предназначен для страхователей, использующих труд членов летных экипажей воздушных судов гражданской авиации для начисления страховых взносов в ПФР по дополнительному тарифу в соответствии с Федеральным законом от 27 ноября 2001 года № 155-ФЗ (в редакции Федерального закона от 24 июля 2009 года № 213-ФЗ). Должности сотрудников, с оплаты труда которых взимаются взносы, должны быть отмечены флажком «Входит в список должностей летных экипажей, работа в которых дает право на доплату к пенсии». Суммы взносов начисляются автоматически в документе «Начисление страховых взносов».

Используется труд занятых на горных работах по добыче угля и сланца - предназначен для страхователей, производящих расчет взносов на выплату доплаты к пенсии работникам организаций угольной промышленности в соответствии с Федеральным законом от 10.05.2010 № 84-ФЗ. Должности сотрудников, с оплаты труда которых взимаются взносы, должны быть отмечены флажком «Дает право на доплату к пенсии в угольной промышленности». Суммы взносов начисляются автоматически в документе «Начисление страховых взносов».

Используется труд членов экипажей морских судов - при включении этой настройки расчет страховых взносов с выплат и иных вознаграждений членам экипажей судов, зарегистрированных в Российском международном реестре судов, будет производиться по пониженному тарифу.

Используется труд фармацевтов - при включении этой настройки расчет страховых взносов с выплат и иных вознаграждений фармацевтам аптечных организаций будет производиться по пониженному тарифу. Должности сотрудников, относящиеся к фармацевтам, должны быть отмечены флажком «Является фармацевтической должностью».

Используется труд работающих в студенческом отряде - этот флаг необходимо использовать в случае, если в программе выполняются выплаты в пользу обучающихся в образовательных учреждениях среднего профессионального, высшего профессионального образования по очной форме обучения за деятельность, осуществляемую в студенческом отряде, в соответствии с пунктом 1 части 3 статьи 9 Федерального закона от 24.07.2009 № 212-ФЗ (в ред. Федерального закона от 28.12.2010 № 428-ФЗ). При взведенном флаге, в форме редактирования данных сотрудника-студента, принятого по трудовому договору, следует взвести флажок «Является студентом работающим в студенческом отряде», или в заключаемом со студентом «Договоре на выполнение работ» - флажок «Заключен со студентом, работающим в студотряде». Начисленные по таким договорам суммы не будут облагаться с 2011 года страховыми взносами в ПФР, а время работы - не будет включаться в страховой стаж для персонифицированного учета ПФР.

НДФЛ

На закладке НДФЛ задается порядок применения стандартных налоговых вычетов при исчислении НДФЛ на очередной налоговый период. Предусмотрено два варианта:

  • Стандартные вычеты применяются нарастающим итогом в течение налогового периода – в этом случае вычеты, на которые имеет право налогоплательщик (сотрудник) с начала года по месяц исчисления налога, применяются к налоговой базе, исчисленной нарастающим итогом за год,
  • Стандартные вычеты применяются в пределах месячного дохода налогоплательщика – в этом случае вычеты, на которые имеет право налогоплательщик (сотрудник) в каждом месяце налогового периода, применяются к налоговой базе, исчисленной за этот месяц (вариант соответствует положениям письма Минфина РФ от 7 октября 2004 г. №03-05-01-04/41).

Допускается изменять порядок применения стандартных налоговых вычетов в течение года, после этого при исчислении НДФЛ за очередной месяц налогового периода будут пересчитаны суммы предоставленных вычетов, а также суммы налога за предыдущие месяцы.

Также на закладке НДФЛ можно настроить вычеты по НДФЛ. Для этого необходимо перейти по гиперссылке «Настроить вычеты по НДФЛ». В отдельно открывшейся форме можно указать:

  • В списке Размеры вычетов по НДФЛ текущие размеры налоговых вычетов;
  • В списке Вычеты к доходам по НДФЛ текущие размеры налоговых вычетов к доходам.

Гос. пособия

На закладке Гос. пособия указываются размеры государственных пособий.

Прочие ставки

На закладке Прочие ставки указываются вспомогательные данные, которые могут использоваться программой:

  • Ставка рефинансирования ЦБ РФ
  • Минимальный размер оплаты труда.

Расчет ЕСН

Закладка видна только при взведенном флаге Показывать устаревшие параметры учета.

На закладке Расчет ЕСН (Единого социального налога) следует указать точность исчисления ЕСН) и взносов в Пенсионный фонд России (ПФР) раздельно по Организациям. Доступны следующие градации точности:

  • С максимальной точностью - до долей копеек,
  • В рублях и копейках,
  • В рублях.

Кроме того, следует указать год, начиная с которого применяются настройки.

Управленческий учет

На закладке Управленческий учет производится настройка параметров управленческого учета.

В поле «Планирование и учет затрат» необходимо указать метод планирования и учета затрат на персонал в управленческом учете. Возможно указать следующие варианты:

  • Метод начислений без выделения страховых взносов (ЕСН);
  • Метод начислений с выделением страховых взносов (ЕСН);
  • Кассовый метод без выделения страховых взносов (ЕСН) и НДФЛ;

  • Кассовый метод с выделением страховых взносов (ЕСН) и НДФЛ.

Кассовый метод отличается тем, что позволяет учитывать в управленческом учете затраты «кассовым методом», т.е. включать в затраты текущего периода выплаты текущего месяца, а также суммы страховых взносов и НДФЛ, начисленные в прошлом месяце, и перечисленные в текущем (предполагается, что страховые взносы и НДФЛ перечисляются по принадлежности своевременно).

Поддерживать несколько схем мотивации - при взведенном флаге будет доступно начисление зарплаты по временной схеме мотивации.

Показывать табельные номера в документах - при выборе этой настройки в формах документов будет показываться колонка с табельным номером сотрудников.

Упрощенный учет взаиморасчетов - флаг необходимо взвести, если используется упрощенный учет взаиморасчетов, т.е когда управленческая зарплата считается выплаченной сразу после начисления, без создания платежных документов.

Расчет зарплаты по ответственным - если взведен этот флаг, то расчет управленческой зарплаты будет проводится по ответственным, где за каждым ответственным закреплено конкретное подразделение. Отредактировать список ответственных можно по ссылке «Показать список ответственных ...».

Количество рабочих дней отпуска в год - устанавливается количество дней отпуска в год для управленческого учета.

Набор персонала ведется - установка режима набора персонала: «по структуре юридических лиц» или «по центрам ответственности».

 

Показывать устаревшие параметры учета - при взведенном флаге на форме отражаются устаревшие данные: закладка Расчет ЕСН, настройка максимального размера пособия (до 2010 года). По умолчанию флаг сброшен.
