Обработка «Формирование документов по НДС»

Обработка используется для формирования регламентных документов по учету НДС.

При выполнении обработки автоматически заполняются и проводятся следующие документы:

  • «Начисление НДС по СМР (хозспособом)»;
  • «Распределение НДС косвенных расходов»;
  • «Восстановление НДС»;
  • «Формирование записей книги покупок» (в том числе, по вычетам при реализации по ставке 0%);
  • «Формирование записей книги продаж» (в том числе, по реализации по ставке 0%);


Формирование документов выполняется по предварительно подготовленной настройке.

Порядок использования обработки:

  • выбрать настройку формирования документов;
  • установить дату формирования документов (документы будут сформированы на последний день месяца, соответствующего дате);
  • нажать кнопку «Сформировать документы».

После формирования документов на закладке «Сформированные документы» отображается список сформированных документов документов. Для каждого документа можно сформировать структуру подчиненности либо движения документа по регистрам.

Над документами из списков можно выполнять групповые операции:

  • проведение и отмену проведение;
  • изменение пометки удаления.
