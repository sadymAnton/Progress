Обработка «Допроведение документов»

Обработка предназначена для допроведения документов.

В форме обработки указывается настройка допроведения документов, в соответствии с которой следует выполнять допроведение.

По настройке определяются параметры допроведения:

  • Список организаций
  • Период допроведения документов
  • Какие будут допроводится документы:
    • либо только требующие допроведения
    • либо все проведенные документы

Непроведенные документы не обрабатываются.

Заполненные по умолчанию параметры допроведения можно изменить:

  • Скорректировать период допроведения
  • Исключить из списка некоторые организации

Есть возможность открыть отчет «Диаграмма отложенного проведения».

Обработка доступна пользователям с ролью «Выполнение регламентных операций (регл. учет)».

См. также механизм «Отложенное проведение».
