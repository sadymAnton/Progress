Регистрация данных о номенклатуре в БД Юнискан

Данная форма позволяет произвести регистрацию данных о номенклатуре информационной базы 1С:Предприятия в Юнискан.

Для выполнения регистрации достаточно выбрать номенклатурную позицию, данные о которой вы желаете зарегистрировать. Поле выбора номенклатуры, программа позволит вам выбрать список единиц измерения для которых вы хотите произвести регистрацию. Для каждой единицы измерения можно зарегистрировать от 0 до 8 транспортных упаковок. Для этого достаточно создать элементы справочника «Транспортные упаковки» для единицы измерения, которую вы желаете зарегистрировать. Регистрация номенклатура в БД Юнискан может быть выполнена в разрезе характеристик. то есть для каждой характеристики может быть зарегистрирована отдельная номенклатурная позиция в БД Юнискан.

Перед выполнением регистрации необходимо проверить правильность указанных для регистрации данных.Важно понимать, что автоматически изменить данные в БД Юнискан нельзя. Для выполнения регистрации достаточно нажать на кнопку «Зарегистрировать». После чего данные будут отправлены в БД Юнискан (регистрация номенклатуры может быть выполнена только в том случае, если компьютер подключен к Интернету).

После выполнения регистрации программа сообщит результат. Каждая зарегистрированная позиция будет подкрашена зеленым цветом, а та позиция, которая не прошла регистрацию - красным.
