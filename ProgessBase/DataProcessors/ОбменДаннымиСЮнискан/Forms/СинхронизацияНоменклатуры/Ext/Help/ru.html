Синхронизация данных о номенклатуре с БД Юнискан

Данная форма позволяет произвести сравнение данных о номенклатуре в информационной базе 1С:Предприятия и данных в информационной базе Юнискан.

Для выполнения сравнения необходимо в таблицу занести штрихкоды информацию, о которых вы желаете получить. Затем нажать на кнопку «Запрос» (Запрос может быть сделан только в том случае, если компьютер подключен к Интернету). После чего будет оправлен запрос к БД Юнискан по указанным штрихкодам.

Для каждого из введенных штрихкодов программа выдаст один из результатов:

·  Не найден в БД Юнискан - означает, что данных о веденном штрихкоде в БД Юнискан нет,

·  Совпадает - означает то, что данные информационной базы 1С:Предприятия и информационной базы Юнискан для данного штрихкода совпадают,

·  Не совпадает - означает то, что данные информационной базы 1С:Предприятия и информационной базы Юнискан для данного штрихкода отличаются.

 

Для того что бы подробнее посмотреть различия данных о номенклатуре в информационной базе 1С:Предприятия и данных в информационной базе Юнискан достаточно позиционироваться на интересующем вас штрихкоде. Различающиеся по значениям поля будут выделены красным цветом.

Для каждого штрихкода, для которого было выполнено сравнение данных с БД Юнискан и для которого в БД Юнискан были найдены данные могут быть выполнены действия:

·  Подобрать номенклатуру. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода не зарегистрирована ни одна номенклатурная позиция. Выполнив это действие можно сопоставить штрихкод номенклатуре, 

·  Создать номенклатуру. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода не зарегистрирована ни одна номенклатурная позиция. Выполнив это действие можно создать новый элемент справочника «Номенклатура» по данным БД Юнискан,

·  Обновить номенклатуру. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода найдена номенклатурная позиция. Выполнив это действие можно изменить реквизиты сопоставленного штрихкоду элемента справочника «Номенклатура». Реквизиты будут изменены согласно данным БД Юнискан.

 

Аналогичные действия могут быть применены и к элементам справочника «Единицы измерения» в том случае если штрихкоду поставлено в соответствие номенклатурная позиция и по данному штрихкоду были найдены данные в БД Юнискан.

·  Подобрать единицу измерения. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода не зарегистрирована ни одна единица измерения. Выполнив это действие можно сопоставить единицу измерения штрихкоду, 

·  Создать единицу измерения. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода не зарегистрирована ни одна единица измерения. Выполнив это действие можно создать новый элемент справочника «Единицы измерения» по данным БД Юнискан,

·  Обновить единицу измерения. Действие доступно только если в информационной базе 1С:Предприятия  для данного штрихкода найдена единица измерения. Выполнив это действие можно изменить реквизиты сопоставленного штрихкоду элемента справочника «Единицы измерения». Реквизиты будут изменены согласно данным БД Юнискан.
