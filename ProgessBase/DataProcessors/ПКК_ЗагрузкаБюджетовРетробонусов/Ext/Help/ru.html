Обработка предназначена для автоматического создания документов "План движения денежных средств" по данным из файлов Excel.

Год - год планирования.

Файл - путь к файлу Excel, содержащему данные для загрузки. Важно! Файл должен иметь название либо "Capex", либо "Opex", в зависимости от выбранного типа.

Сценарий - сценарий планирования создаваемых документов.

______________________________________________________________________________________________

По нажатию на кнопку "Выполнить" создаются документы с заполненными реквизитами*:

Сценарий - выбранный сценарий в обработке.

Подразделение - элемент справочника "Подразделения" найденный по наименованию из файла.

Дата планирования - первое число месяца при Opex`e и первое число квартала при Capex`e.

Валюта - рубли.

Вид денежных средств - безнал.

Capex/Opex - значение выбранное в обработке.

Заполняемые реквизиты табличной части:

Вид движения - расход.

Статья ДДС - элемент справочника "Статьи движения денежных средств", найденный по "ФинКоду" из файла.

Проект - заполняется проектом из файла при Capex`e.

Сумма - значение из файла умноженное на 1000.

 

*Создается по документу на каждое подразделение и период планирования. Т.е. если в файле одно подразделение, то создастся 12 документов по подразделению на каждый месяц планирования (Opex) или 4 документа, если Capex.
