Настройка программы "Использовать регистр "Свободные остатки""

Контроль остатков товаров на складах может выполняться по регистру "Свободные остатки". Этот регистр содержит сводные данные по регистрам "Товары на складах", "Товары в рознице", "Товары в резерве на складах", "Товары к передаче со складов" (далее базовые регистры).

Включение режима "Использовать регистр "Свободные остатки""

Изменение настройки выполняется через интерфейс "Заведующий учетом", меню "Настройка учета" - "Настройка программы", раздел "Контроль остатков".

При выборе "Использовать регистр "Свободные остатки"" требуется ввести дату, начиная с которой регистр будет заполнен данными базовых регистров. Например, при вводе даты 01.01.2007 регистр "Свободные остатки" будет содержать остатки на 01.01.2007 и движения, начиная с этой даты. При выборе даты необходимо учитывать, что последующие изменения данных до этой даты могут привести к рассогласованию регистра.

При подтверждении использования регистра "Свободные остатки" будут выполнены следующие действия:

  • Если в информационной базе существуют движения по регистру "Свободные остатки", эти движения будут удалены

  • Будут записаны остатки документом "Корректировка записей регистров" на выбранную дату.

    Если режим устанавливается не первый раз, то рекомендуется предварительно удалить созданные ранее документы по вводу остатков.

  • Если в системе есть документы, по которым были сформированы движения по регистрам-основаниям более поздней датой, чем дата, начиная с которой заполняется регистр "Свободные остатки", то по этим документам будут записаны движения в регистр "Свободные остатки".

При отключении использования регистра "Свободные остатки" изменения данных регистра "Свободные остатки" не происходит.

Поддержка регистра в согласованном состоянии

Согласованность данных регистра проверяется с помощью отчета Анализ регистра "Свободные остатки".

Следующие действия могут привести к рассогласованию данных в регистре "Свободные остатки" с данными базовых регистров.

Действия

Способ предотвращения

Изменение базовых регистров датой меньшей, чем дата начала использования регистра "Свободные остатки". Запретить изменение базовых регистров до введенной даты. Например, при помощи установки даты запрета изменения данных.
Изменение базовых регистров при помощи документа "Корректировка записей регистров".

При корректировке базовых регистров изменять также регистр "Свободные остатки". После корректировки рекомендуется проверить согласованность данных.

Использование обработки "Поиск и замена дублирующихся элементов справочников" без включенной настройки "Отключать контроль записи". Включить настройку "Отключать контроль записи".
Если это невозможно, то после выполнения обработки выполнить повторное включение режима "Использовать регистр "Свободные остатки"".
Использование внешних обработок, которые могут изменять базовые регистры.

Выполните проверку согласованности данных регистра.
Если данные не согласованы, то выполните повторное включение режима "Использовать регистр "Свободные остатки"".
