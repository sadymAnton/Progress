Настройки маршрутов согласования заявок 

Предназначен для настройки маршрутов согласования заявок на расходование денежных средств.

Маршрут согласования указывается для подразделения. Если маршрут согласования состоит из нескольких этапов, в настройке необходимо указать начальный этап согласования. По мере прохождения каждого этапа процесс согласования будет переходить на вышестоящий по иерархии этап согласования.

Маршрут согласования заявки на расходование денежных средств определяется по указанному в заявке Подразделению.

Если маршрут не назначен указанному подразделению - используется маршрут, указанный для пустого подразделения.
