Регистр сведений «Номенклатура для реализации без НДС и НДС 0%»

Предназначен для указания номенклатуры готовой продукции, работ и услуг, которые реализуются (или могут быть реализованы в будущем) без НДС или с применением ставки НДС 0%, при условии, что производятся также операции по реализации с «обычными» ставки НДС (18%, 10% и т.д.). В регистр также следует вносить сведения обо всех полуфабрикатах собственного изготовления, которые используются для производства такой продукции.

Исходя из своего назначения, данный регистр используется только в случаях, когда в учетной политике организации по налоговому учету предусмотрено ведение сложного учета «входного» НДС (в регистре сведений «Учетная политика (бухгалтерский и налоговый учет)» установлен флаг «Организация осуществляет реализацию без НДС или с НДС 0%»).

Записи в регистр производятся с помощью документа «Выбор номенклатуры для ведения сложного учета по НДС».

В регистре содержится всего три показателя - организация, номенклатура и условие ведения «сложного» учета по НДС (в виде «Да/Нет»). Последнее необходимо, чтобы ведение сложного учета НДС для определенной номенклатуры продукции, работ или услуг можно было прекратить с некоторого момента, когда в этом уже нет необходимости (например, когда данная продукция реализуется целиком на внутреннем рынке с НДС 18%, и ее экспорт в будущем не планируется, хотя в прошлом имел место).
